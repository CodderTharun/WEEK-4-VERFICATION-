# Machine Design - Week 3

## Springs

### Spring Types
- Helical compression springs
- Helical tension springs
- Torsion springs
- Leaf springs

### Spring Constant
- k = Force/Deflection
- Series: 1/k_eq = 1/k₁ + 1/k₂
- Parallel: k_eq = k₁ + k₂

## Vibrations

### Free Vibration
- Natural frequency: ωₙ = √(k/m)
- Damping ratio: ζ = c/(2√(km))

### Forced Vibration
- Resonance at ωₙ
- Transmissibility

### Isolation
- Vibration dampers
- Isolators
