## Computer Networks

| Week | Syllabus |
| --- | --- |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 1</strong></span> | Introduction and Physical Layer: Definition of Computer Networks and Advantages. Network Devices: Router, Switch, Hub, Bridge, Gateway, Access Point, Modem, Firewall. Types of Networks: PAN, LAN, CAN, MAN, WAN. Internet and Protocols. OSI Model and TCP/IP Model. Network Topologies: Bus, Star, Ring, Mesh and Hybrid. Transmission Modes: Simplex, Half-Duplex and Full-Duplex. Transmission Media: Guided and Unguided Media. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 2</strong></span> | Data Link Layer: Services, Basic Design Issues and Framing Techniques. Framing Methods: Byte Count, Bit Stuffing and Byte Stuffing. Error and Flow Control: Error Detection using Parity, CRCs and Checksum. Error Correction: Hamming Codes, Binary Convolutional Codes, Reed-Solomon Codes and Low-Density Parity Check Codes. Sliding Window Protocols: Go-Back-N and Selective Repeat. Medium Access Sublayer: Channel Allocation Problems, MAC Protocols, ALOHA, Carrier Sense Protocols and Collision-Free Protocols. |

## Operating Systems

| Week | Syllabus |
| --- | --- |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 1</strong></span> | Introduction and Process Concepts, Basics: What is OS?, OS Operations, OS Design and Implementation. Process: Process Scheduling, Process Operations (Life Cycle), IPC and Multithreading Processing, Threads and Types of Threading Models, Process Scheduling Algorithms. Synchronization: Critical Section Problem, Peterson’s Solution, Synchronization Hardware and Semaphores. Synchronization Problems: Bounded, Buffer Problem, Readers-Writers Problem and Dining-Philosophers Problem. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 2</strong></span> | Deadlocks: What are Deadlocks and Issues caused by them. Deadlock Conditions: Mutual Exclusion, Hold and Wait, No Preemption and Circular Wait. Deadlock Avoidance: Safe State, Resource Allocation Graph Algorithm, Banker’s Algorithm, Safety Algorithm and Resource Request Algorithm. Deadlock Detection: Single Instance and Multiple Instances of Resource Types. Recovery from Deadlock: Process Termination and Resource Preemption. |
