## Computer Networks

| Week | Syllabus |
| --- | --- |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 1</strong></span> | Introduction and Physical Layer: Definition of Computer Networks and Advantages. Network Devices: Router, Switch, Hub, Bridge, Gateway, Access Point, Modem, Firewall. Types of Networks: PAN, LAN, CAN, MAN, WAN. Internet and Protocols. OSI Model and TCP/IP Model. Network Topologies: Bus, Star, Ring, Mesh and Hybrid. Transmission Modes: Simplex, Half-Duplex and Full-Duplex. Transmission Media: Guided and Unguided Media. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 2</strong></span> | Data Link Layer: Services, Basic Design Issues and Framing Techniques. Framing Methods: Byte Count, Bit Stuffing and Byte Stuffing. Error and Flow Control: Error Detection using Parity, CRCs and Checksum. Error Correction: Hamming Codes, Binary Convolutional Codes, Reed-Solomon Codes and Low-Density Parity Check Codes. Sliding Window Protocols: Go-Back-N and Selective Repeat. Medium Access Sublayer: Channel Allocation Problems, MAC Protocols, ALOHA, Carrier Sense Protocols and Collision-Free Protocols. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 3</strong></span> | Network Layer: Services and Routing Algorithms. Routing Techniques: Shortest Path Routing, Flooding, Distance Vector Routing, Link State Routing and Hierarchical Routing. Broadcast and Anycast Routing. Congestion Control Techniques: Traffic Aware Routing, Admission Control, Traffic Throttling and Load Shedding. Quality of Service (QoS): Traffic Shaping, Packet Scheduling and Admission Control. Internet Protocols and Services (IS and DS). IP Protocols: IPv4 and IPv6. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 4</strong></span> | Transport Layer: Services and Elements of Transport Layer. Addressing, Connection Establishment and Connection Release. Congestion Control Techniques, Bandwidth Allocation, Max-Min Fairness, Convergence and Sending Rate Mechanisms. Transport Layer Protocols: UDP and TCP. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 5</strong></span> | Application Layer: Client–Server Model, Cookies and Sessions, SSL/TLS Basics. Application Layer Protocols: HTTP/HTTPS, DNS, FTP, SMTP, POP3 and IMAP. |

## Operating Systems

| Week | Syllabus |
| --- | --- |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 1</strong></span> | Introduction and Process Concepts. Basics: What is OS?, OS Operations, OS Design and Implementation. Process: Process Scheduling, Process Operations (Life Cycle), IPC and Multithreaded Processing. Threads and Types of Threading Models. Process Scheduling Algorithms. Multiple-Processor Scheduling: Approach, Issues, Process Affinity and Load Balancing. Synchronization: Critical Section Problem, Peterson’s Solution, Synchronization Hardware and Semaphores. Synchronization Problems: Bounded-Buffer Problem, Readers-Writers Problem and Dining-Philosophers Problem. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 2</strong></span> | Deadlocks: What are Deadlocks and Issues caused by them. Deadlock Conditions: Mutual Exclusion, Hold and Wait, No Preemption and Circular Wait. Deadlock Avoidance: Safe State, Resource Allocation Graph Algorithm, Banker’s Algorithm, Safety Algorithm and Resource Request Algorithm. Deadlock Detection: Single Instance and Multiple Instances of Resource Types. Recovery from Deadlock: Process Termination and Resource Preemption. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 3</strong></span> | Memory Management: Swapping, Contiguous Memory Allocation and Fragmentation. Paging: Methodologies and Structures of the Page Table. Segmentation and Virtual Memory Management. Demand Paging and Copy-on-Write. Page Replacement and Types of Page Replacement Algorithms: FIFO, Optimal and Least Recently Used (LRU), LRU Approximation. Allocation of Frames and Allocation Algorithms. Thrashing. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 4</strong></span> | File System: File Concept, File Attributes, File Operations and File Types. Access Methods: Direct, Sequential and Index-Based Access. Directory Structure: Single-Level Directory, Two-Level Directory, Tree-Structured Directories, Acyclic Graph Directories and General Graph Directory. File Sharing and Protection. File System Implementation: Partition and Mounting, Virtual File Systems. Directory Implementation: Linear List and Hash Table. Allocation Methods: Contiguous Allocation, Linked Allocation and Indexed Allocation. Free Space Management: Bit Vector, Linked List, Grouping and Counting. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 5</strong></span> | Secondary Storage Management. Disk Scheduling: FCFS, SSTF, SCAN, C-SCAN, LOOK Scheduling and C-LOOK Scheduling. Disk Management: Disk Formatting, Boot Block and Bad Blocks. System Protection and Access Matrix. |
