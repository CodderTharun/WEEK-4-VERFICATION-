## Computer Networks

| Week | Syllabus |
| --- | --- |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 1</strong></span> | Introduction and Physical Layer: Definition of Computer Networks and Advantages. Network Devices: Router, Switch, Hub, Bridge, Gateway, Access Point, Modem, Firewall. Types of Networks: PAN, LAN, CAN, MAN, WAN. Internet and Protocols. OSI Model and TCP/IP Model. Network Topologies: Bus, Star, Ring, Mesh and Hybrid. Transmission Modes: Simplex, Half-Duplex and Full-Duplex. Transmission Media: Guided and Unguided Media. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 2</strong></span> | Functions of Data Link Layer. Link Layer Addressing: Unicast, Multicast and Broadcast. Address Resolution Protocol (ARP): Message Types, ARP Operation, ARP Packet and Reverse ARP (RARP). Medium Access Control Protocols: CSMA (Carrier Sense Multiple Access), CSMA/CD, CSMA/CA and ALOHA Protocols (Pure ALOHA and Slotted ALOHA). Ethernet and Ethernet Frames: Key Features. Error Detection in Computer Networks: Single-Bit Error, Multiple-Bit Error, Burst Error, Error Detection Methods – Simple Parity Check, Two-Dimensional Parity Check, Checksum and Cyclic Redundancy Check (CRC). Error Correction in Computer Networks: Forward Error Correction, Backward Error Correction and Hamming Code. Hamming Code Operations: Encoding a Message and Decoding a Message. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 3</strong></span> | Computer Networks – Network & Transport Layers: Functions of the Network Layer, How the Network Layer Works, Advantages and Limitations of the Network Layer, Services Offered by the Network Layer, Forwarding, IP Addressing, How IP Addresses Work, IPv4, IPv6, Real-World Scenario: Sending an Email from New York to Tokyo, Introduction to Subnetting, Role of Subnet Masks, Calculating the Number of Hosts in a Subnet, Classless Inter-Domain Routing (CIDR), Routing Fundamentals, Working Principle of Routing, Routing Tables, Distance Vector Routing (DVR) Protocol, Transport Layer, TCP 3-Way Handshake, TCP Connection Establishment, TCP Connection Termination (Graceful and Abrupt), and Congestion Control in Computer Networks. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 4</strong></span> | Transport Layer: Services and Elements of the Transport Layer, Addressing, Connection Establishment and Connection Release. Congestion Control Techniques, Bandwidth Allocation, Max-Min Fairness, Convergence and Sending Rate Mechanisms. Transport Layer Protocols: UDP and TCP. |

## Database Management Systems

| Week | Syllabus |
| --- | --- |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 1</strong></span> | Introduction, More about DBMS: Characteristics, Advantages and Disadvantages, Database System Concepts: Database Models (Relational, ER, Object based, Hierarchical and Semistructured), Database Architecture (1-Tier, 2-Tier and 3-Tier), Relational Model. Three-Schema Architecture: Internal Schema, Conceptual Schema, External Schema. Database languages: DDL, DML, DCL and TCL, ER Model: Notations, Attributes, Keys, Relationships, Cardinality and Participation constraint.|
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 2</strong></span> | Types of DBMS: Hierarchical DBMS, Network DBMS, Relational DBMS (RDBMS) and Object-Oriented DBMS. ER to Relational Mapping: Entity Mapping, Attribute Mapping and Relationship Mapping. Relational Model Concepts: Relation, Tuple, Attribute, Domain, Degree and Cardinality. Relational Algebra Operations: Unary Operations – SELECT and PROJECT. Binary Operations – UNION and INTERSECTION. Additional Operation – RENAME. Advanced Operations – JOIN and DIVISION. SQL Basics: Introduction to SQL, SQL Commands and SQL Queries. SQL Data Types: INT, VARCHAR, CHAR, FLOAT, DATE and BOOLEAN. SQL Operators: AND, OR, NOT, BETWEEN, IN, LIKE and IS NULL. SQL JOINS: INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN, NATURAL JOIN, CROSS JOIN and SELF JOIN. Aggregate Functions in SQL: COUNT(), SUM(), AVG(), MAX() and MIN(). |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 3</strong></span> | Advanced SQL & PL/SQL: UNION and UNION ALL, SELECT INTO Statement, CREATE TABLE, SQL Indexing and Performance Optimization, SQL Composite Indexes, Query Processing (Parsing, Optimization, and Evaluation), EXPLAIN and Query Plans, Views, Introduction to PL/SQL, Variables and Data Types, Conditional Statements, Loops, Exception Handling, Stored Procedures, and Functions. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 4</strong></span> | Database Normalization: Introduction to Normalization, Need, Features and Advantages of Normalization, Database Anomalies (Insertion, Deletion and Update Anomalies), Functional Dependency, First Normal Form (1NF), Second Normal Form (2NF), Third Normal Form (3NF), Boyce-Codd Normal Form (BCNF), Fourth Normal Form (4NF), Fifth Normal Form (5NF) and Practical Examples. |

## Operating Systems

| Week | Syllabus |
| --- | --- |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 1</strong></span> | Introduction and Process Concepts, Basics: What is OS?, OS Operations, OS Design and Implementation. Process: Process Scheduling, Process Operations (Life Cycle), IPC and Multithreading Processing, Threads and Types of Threading Models, Process Scheduling Algorithms. Synchronization: Critical Section Problem, Peterson’s Solution, Synchronization Hardware and Semaphores. Synchronization Problems: Bounded, Buffer Problem, Readers-Writers Problem and Dining-Philosophers Problem. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 2</strong></span> | Deadlocks: What are Deadlocks and Issues caused by them. Deadlock Conditions: Mutual Exclusion, Hold and Wait, No Preemption and Circular Wait. Deadlock Avoidance: Safe State, Resource Allocation Graph Algorithm, Banker’s Algorithm, Safety Algorithm and Resource Request Algorithm. Deadlock Detection: Single Instance and Multiple Instances of Resource Types. Recovery from Deadlock: Process Termination and Resource Preemption. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 3</strong></span> | Memory Management: Swapping, Contiguous Memory Allocation, and Fragmentation. Paging: Methodologies and Page Table Structures. Segmentation and Virtual Memory Management. Demand Paging and Copy-on-Write. Page Replacement Techniques and Algorithms including FIFO, Optimal, Least Recently Used (LRU), and LRU Approximation. Frame Allocation Strategies and Allocation Algorithms. Thrashing and its Prevention Techniques. |
| <span style={{ whiteSpace: 'nowrap' }}><strong>Week 4</strong></span> | File System: File Concept, File Attributes, File Operations and File Types. Access Methods: Direct, Sequential and Index-Based Access. Directory Structure: Single-Level, Two-Level, Tree-Structured, Acyclic Graph and General Graph Directories. File Sharing and Protection. File System Implementation: Partitioning, Mounting and Virtual File Systems. Directory Implementation: Linear List and Hash Table. Allocation Methods: Contiguous, Linked and Indexed Allocation. Free Space Management: Bit Vector, Linked List and Free List. |
