## Sorting Algorithms

Sorting is arranging data in a specific order.

### Common Sorting Algorithms

1. **Bubble Sort** - O(n²) time complexity
   - Simple but inefficient
   - Compare adjacent elements

2. **Selection Sort** - O(n²) time complexity
   - Select minimum and place it

3. **Insertion Sort** - O(n²) average, O(n) best case
   - Build sorted array one item at a time

4. **Merge Sort** - O(n log n) time complexity
   - Divide and conquer
   - Stable sort

5. **Quick Sort** - O(n log n) average, O(n²) worst
   - Efficient for large datasets
   - In-place sorting

### Searching Algorithms

1. **Linear Search** - O(n) time complexity
   - Simple sequential search

2. **Binary Search** - O(log n) time complexity
   - Works on sorted arrays
   - Divide and conquer
