## TCP/IP Model

The TCP/IP model is the foundation of modern internet.

### TCP/IP Layers

1. **Application Layer**
   - HTTP, HTTPS, FTP, SMTP, DNS

2. **Transport Layer**
   - TCP, UDP

3. **Internet Layer**
   - IP, ICMP, IGMP

4. **Link Layer**
   - Ethernet, PPP, Wi-Fi

### TCP (Transmission Control Protocol)

- Connection-oriented
- Reliable delivery
- In-order delivery
- Flow control
- Used for: Email, Web, File transfer

### UDP (User Datagram Protocol)

- Connectionless
- Unreliable but fast
- No flow control
- Used for: Video streaming, Online gaming, DNS

### IP Addressing

- **IPv4**: 32-bit address (xxx.xxx.xxx.xxx)
- **IPv6**: 128-bit address
- **Subnetting**: Dividing networks
