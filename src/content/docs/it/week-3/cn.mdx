## Routing

Routing is the process of forwarding data packets from source to destination.

### Routing Concepts

- **Router** - Device that forwards packets
- **Routing Table** - Table at each router with routes
- **Hop** - Each router a packet passes through
- **TTL** - Time To Live (prevents infinite loops)

### Routing Protocols

1. **Distance Vector Routing**
   - RIP (Routing Information Protocol)
   - EIGRP (Enhanced IGRP)

2. **Link State Routing**
   - OSPF (Open Shortest Path First)
   - IS-IS

3. **Path Vector Routing**
   - BGP (Border Gateway Protocol)

### IP Routing

- Routing based on destination IP address
- Longest prefix match
- Default gateway

### Network Layers Review

- **Physical Layer** - Transmission medium
- **Data Link Layer** - MAC addressing, switching
- **Network Layer** - IP routing, logical addressing
