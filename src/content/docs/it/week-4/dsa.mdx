## Graphs

A graph is a collection of vertices (nodes) and edges (connections).

### Graph Types

1. **Directed Graph** - Edges have direction
2. **Undirected Graph** - Edges have no direction
3. **Weighted Graph** - Edges have weights
4. **Cyclic/Acyclic** - Contains cycles or not

### Graph Representation

1. **Adjacency Matrix** - 2D array
   - Space: O(V²)
   - Dense graphs efficient

2. **Adjacency List** - Array of lists
   - Space: O(V + E)
   - Sparse graphs efficient

### Graph Traversal

1. **Depth-First Search (DFS)**
   - Uses stack or recursion
   - Time: O(V + E)

2. **Breadth-First Search (BFS)**
   - Uses queue
   - Time: O(V + E)
   - Finds shortest path in unweighted graph
