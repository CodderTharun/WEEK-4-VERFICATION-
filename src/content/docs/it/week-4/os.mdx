# Operating Systems - Week 4

## File Systems

File system manages how data is stored and retrieved.

### File System Components

- **File** - Named collection of data
- **Directory** - Container for files and subdirectories
- **Partition** - Logical division of disk
- **Volume** - Mountable file system

### File Organization

1. **Sequential File** - Records in order
2. **Direct File** - Accessed by key
3. **Indexed Sequential** - Index for access

## Disk Scheduling

Determines order of disk access requests.

### Disk Scheduling Algorithms

1. **FCFS** - First Come First Served
   - Simple but inefficient

2. **SSTF** - Shortest Seek Time First
   - Minimizes head movement

3. **SCAN (Elevator Algorithm)**
   - Head moves back and forth
   - Fair and efficient

4. **C-SCAN** - Circular SCAN
   - Head returns to end after reaching limit

### Disk Allocation Methods

- **Contiguous** - Blocks allocated contiguously
- **Linked** - Linked list of blocks
- **Indexed** - Index blocks
