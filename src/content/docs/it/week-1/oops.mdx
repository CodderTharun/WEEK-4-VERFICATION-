# Object-Oriented Programming - Week 1

## OOPS Fundamentals

Object-Oriented Programming is a programming paradigm based on objects and classes.

### Core Concepts

1. **Class** - Blueprint for creating objects
2. **Object** - Instance of a class
3. **Encapsulation** - Bundling data and methods
4. **Inheritance** - Derived classes from base classes
5. **Polymorphism** - Same interface, different implementations
6. **Abstraction** - Hiding complex details

### Class Structure

```python
class Car:
    def __init__(self, brand, model):
        self.brand = brand
        self.model = model
    
    def display(self):
        return f"{self.brand} {self.model}"
```

### Basic Principles

- **Modularity** - Code organized in modules
- **Reusability** - Code can be reused
- **Maintainability** - Easy to modify and update
- **Flexibility** - Adaptable to changes
