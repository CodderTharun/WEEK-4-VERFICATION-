## Python

Python is a high-level, interpreted programming language known for its simplicity and versatility. It is widely used in various fields such as artificial intelligence, web development, automation, and data analysis.

Its easy-to-read syntax makes it an excellent choice for beginners and experienced programmers alike.

Understanding the fundamental concepts of Python is essential for effective programming. These basics include:

- Variables and data types
- Input/output operations
- Operators
- Control flow statements
- Functions
- Data structures

### D. Control Flow

#### 1. Conditional Statements

Conditional statements allow the program to make decisions based on certain conditions.

The main structures are:

- **if** – executes code if condition is true
- **elif** – checks additional conditions
- **else** – executes if none of the above conditions are true

#### 2. Loops

Loops enable repeated execution of code blocks.

Python provides:

- **for** – iterates over a sequence or range
- **while** – repeats as long as a condition is true

Control statements like `break`, `continue`, and `pass` modify loop behavior.

Python uses indentation (usually 4 spaces) instead of curly braces `{}` to define blocks of code.

#### 3. Conditional Statements Example

<Tabs>

## NumPy

NumPy (Numerical Python) is a fundamental Python library used for numerical computing.  
It provides support for arrays, matrices, and a wide range of mathematical functions that operate efficiently on these data structures.

NumPy is especially useful in scientific computing and data analysis because it enables high-performance operations on large multidimensional arrays and matrices.

Although NumPy arrays look similar to standard Python lists, they are significantly faster and more memory-efficient. Its broadcasting feature is also highly useful, as it allows mathematical operations to be applied to entire datasets quickly and efficiently.

---


### A. Key Features of NumPy

#### 1. N-dimensional Array (ndarray)

NumPy’s primary data structure is the ndarray, which allows efficient storage and manipulation of large datasets in multiple dimensions such as 1D, 2D, and 3D arrays.

#### 2. Mathematical Functions

NumPy provides a wide range of mathematical operations such as:

- Linear algebra functions
- Statistical functions
- Random number generation

#### 3. Broadcasting

Broadcasting allows NumPy to perform element-wise operations on arrays of different shapes without explicitly reshaping them.

#### 4. Integration with Other Libraries

NumPy is commonly used alongside:
- Pandas
- SciPy
- Matplotlib
It forms the foundation for scientific computing in Python.

#### 5. Efficiency

NumPy operations are faster and more memory-efficient compared to traditional Python lists because they are implemented using lower-level languages.

## Pandas

Pandas is the most popular Python library for data analysis.
It acts as a bridge between raw data such as CSVs or SQL databases and the mathematical power of NumPy.
Pandas introduces two primary data structures:
- Series (1D)
- DataFrame (2D)

These structures allow you to manipulate tabular data with the same ease as an Excel spreadsheet but with the speed and scalability of Python.

### A.  Core Data Structures

#### 1. Pandas Series

A Series is a one-dimensional labeled array.
It is essentially a single column of data where every value is assigned a label called an index.

##### a. Key Feature

You can access data using labels instead of just numerical positions.

<Tabs>
  <TabItem label="Python">

```python
import pandas as pd

# Creating a Series from a list
ser = pd.Series([10, 20, 30], index=['a', 'b', 'c'])

print(ser['b'])
# Accessing by label returns 20
```

  </TabItem>
</Tabs>

#### 2. Pandas DataFrame

A DataFrame is a two-dimensional, size-mutable, and heterogeneous tabular data structure.
Think of it as a collection of Series objects sharing the same index.

##### a. Key Feature

It can handle different data types such as:
- Integers
- Strings
- Floats
- Booleans

in different columns simultaneously.

<Tabs>
  <TabItem label="Python">

```python
data = {
    'Product': ['Laptop', 'Mouse', 'Monitor'],
    'Price': [1200, 25, 300],
    'In_Stock': [True, True, False]
}

df = pd.DataFrame(data)
```

  </TabItem>
</Tabs>

### B.  Essential Data Operations

#### 1. Data Exploration

When a dataset is loaded, it is important to inspect its contents without printing thousands of rows.

Common methods include:
- `df.head(n)` → Displays the first n rows
- `df.info()` → Shows data types, memory usage, and null counts
- `df.describe()` → Provides a statistical summary of numerical columns

#### 2. Data Cleaning

Real-world data is often missing or incorrectly formatted.

##### Handling Missing Data

Use:
- `df.isnull()` to identify missing values
- `df.fillna(value)` to replace missing values
- `df.dropna()` to remove missing data

- Left Join

##### b. Grouping (Split-Apply-Combine)

## Python - Matplotlib

Matplotlib is a powerful and versatile open-source plotting library for Python that helps users visualize data in a variety of formats.  

It is designed to work seamlessly with NumPy arrays and pandas DataFrames, making it highly useful for scientific computing and data analysis.  

The library provides an easy way to create graphs and charts from tabular data, including line plots, bar charts, histograms, scatter plots, and more.

---

### A. Matplotlib Pyplot

Pyplot is a Matplotlib module that provides simple functions for adding plot elements, such as lines, images, text, etc. to the axes in the current figure.

### B. Steps to Use Pyplot:

1. **Import Matplotlib:** Start by importing matplotlib.pyplot as plt.
2. **Create Data:** Prepare your data in the form of lists or arrays.

### F. Create Legends

A matplotlib legend is a small box that describes the various elements of the plot in terms of lines and markers that have distinct styles or colors.

**Attributes:**

a. **shadow:** [None or bool] Whether to draw a shadow behind the legend.It’s Default value is None.

b. **markerscale:** [None or int or float] The relative size of legend markers compared with the originally drawn ones.The Default is None. 

c. **numpoints:** [None or int] The number of marker points in the legend when creating a legend entry for a Line2D (line).The Default is None. 

d. **fontsize:** The font size of the legend.If the value is numeric the size will be the absolute font size in points. 

e. **facecolor:** [None or “inherit” or color] The legend’s background color. 

f. **edgecolor:** [None or “inherit” or color] The legend’s background patch edge color.

`#adding legend in graph => plt.legend(x, y, label='legend');
#plt.legend() will use`

<Tabs>
  <TabItem label="Python">

```python
x = list(range(0, 5))
y = list(range(0, 10, 2))

fig = plt.figure(figsize=(3, 3))
ax = fig.add_subplot(1, 1, 1)

ax.set(
    title="Figure",
    xlabel="x-axis",
    ylabel="y-axis",
    xlim=(0, 5),
    ylim=(0, 10)
)

plt.plot(x, y, label="legend")

plt.legend()
plt.show()
```

## Plotting in Matplotlib

### A. Line Plot

Line Plot is used to visualize a trend in data and is also used to compare two variables.

**Syntax:** `plt.plot(x,y)`

<Tabs>
  <TabItem label="Python">

```python
x = [1, 5, 8, 12, 15, 19, 22, 26, 29]
y = [29.3, 30.1, 30.4, 31.5, 32.3, 32.6, 31.8, 32.4, 32.7]

fig = plt.figure(figsize=(8, 6))
ax = fig.add_subplot(1, 1, 1)

ax.set(
    title="Line Plot Graph",
    xlabel="X-Axis",
    ylabel="Y-Axis",
    xlim=(0, 30),
    ylim=(25, 35)
)

ax.plot(x, y)

plt.show()
```

#### Parameter for Plot

1. **color** - Sets the color of the line.
2. **linestyle** - Sets the line style, e.g., solid, dashed, etc.
3. **linewidth** - Sets the thickness of a line.
4. **marker** - Chooses a marker for data points, e.g., circle, triangle, etc.
5. **markersize** - Sets the size of the chosen marker.
6. **label** - Names the line, which will come in legend.

<Tabs>
  <TabItem label="Python">

```python
fig = plt.figure(figsize=(6, 3))
ax = fig.add_subplot(1, 1, 1)

ax.set(
    title="Line Plot Graph",
    xlabel="X-Axis",
    ylabel="Y-Axis",
    xlim=(0, 30),
    ylim=(25, 35)
)

ax.plot(
    x,
    y,
    color="black",
    linestyle="dotted",
    linewidth=1,
    marker="o",
    markersize=10,
    label="label"
)

plt.legend()
plt.show()
```

### B. Scatter Graph

It similar to a line graph which is used to show how one variable is related to another. It consists of data points, if it is linear then it is highly correlated and only marks the data point.

**Syntax:** `plt.scatter(x,y)`

#### Parameter of Scatter Graph

1. **c:** Sets color of markers.
2. **s:** Sets the size of markers.
3. **marker:** Select a marker. e.g.: circle, triangle, etc
4. **edgecolor:** Sets the color of lines on the edges of markers.

<Tabs>
  <TabItem label="Python">

```python
x = [1, 4, 6, 4, 2]
y = [10, 10, 1, 5, 0]

fig = plt.figure(figsize=(8, 4))
ax = fig.add_subplot()

ax.scatter(
    x,
    y,
    c="blue",
    s=500,
    marker="o",
    edgecolor="black",
    label="Scatter"
)

ax.set_title("Scatter Graph")
ax.set_xlabel("X-Axis")
ax.set_ylabel("Y-Axis")

plt.legend()
plt.show()
```

### C. Bar Chart

It is mostly used to compare categories, for vertical bar plots and horizontal bar plots.

**Syntax:** `bar(x, height) or bar(y,width)`

#### Parameter of Bar Graph

1. **Color:** Sets the color of bars.
2. **edgecolor:** Sets the color of the borderline of bars.
3. **label:** Sets label to a bar, appearing in legend.
4. **color:** The color of the bar.
5. **align:** Aligns the bars with respect to x-coordinates.

<Tabs>
  <TabItem label="Vertical Bar Graph">

```python
# Vertical Bar Graph
import matplotlib.pyplot as plt

x = [1, 2, 3]
y = [10, 20, 30]

fig = plt.figure(figsize=(8, 4))
ax = fig.add_subplot()

ax.set(
    title="Bar Graph",
    xlabel="X-Axis",
    ylabel="Y-Axis",
    xticks=x,
    label="label",
    xticklabels=["first", "second", "third"]
)

ax.bar(
    x,
    y,
    color="blue",
    edgecolor="black",
    width=0.5,
    align="edge",
    label="label",
    linewidth=3
)

plt.legend()
plt.show()
```

  </TabItem>

  <TabItem label="Horizontal Bar Graph">


### D. Pie Plot

It is effective in showing the proportion of categories and is best suited for comparing fewer categories. It is used to highlight the proportion of one or a group of categories.

**Syntax:** pie(x), x: size of portions, passed as fraction or number

#### Parameter of Pie

1. **colors:** Sets the colors of portions.
2. **labels:** Sets the labels of portions.
3. **startangle:** Sets the start angle at which the portion drawing starts.
4. **autopct:** Sets the percentage display format of an area, covering portions.

<Tabs>
  <TabItem label="Python">

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]

fig = plt.figure(figsize=(4, 4))
ax = fig.add_subplot()

ax.set(title="Pie Chart")

plt.pie(
    x,
    colors=["brown", "red", "green", "yellow", "blue"],
    labels=["first", "second", "third", "fourth", "fifth"],
    startangle=0,
    autopct="%1.1f%%"
)

plt.show()
```

### E. Histogram Chart

It is used to visualize the spread of data in a distribution

**Syntax:** hist(x)

where x is the data values

#### Parameter of Histogram

1. **Color:** Sets the color of bars.
2. **bins:** Sets the number of bins to be used.
3. **density:** Sets to True where bins display fraction and not the count.

<Tabs>
  <TabItem label="Python">

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]

fig = plt.figure(figsize=(10, 4))
ax = fig.add_subplot()

ax.set(
    title="Histogram",
    xlabel="X-Axis",
    ylabel="Y-Axis"
)

ax.hist(
    x,
    color="blue",
    bins=10,
    density=True,
    orientation="vertical"
)

plt.show()
```

### A. Display an Image in Grayscale in Matplotlib

Open the image using PIL image method and convert it to L mode.L mode image is a single-channel image - normally interpreted as grayscale.

<Tabs>
  <TabItem label="Python">

```python
from PIL import Image
import matplotlib.pyplot as plt

# Storing image path
fname = r'/content/image.jpg'

# Opening image using PIL and converting to grayscale
image = Image.open(fname).convert("L")

# Displaying the grayscale image
plt.imshow(image, cmap='gray')

# Optional: hides axis ticks
plt.axis('off')

plt.show()
```

### B. Display an OpenCV image in Python with Matplotlib

The OpenCV module is an open-source computer vision and machine learning software library. It is a huge open-source library for computer vision, machine learning, and image processing.

OpenCV images with Matplotlib module for that you just need to convert colored image into a gray scale image.

<Tabs>
  <TabItem label="Python">

```python
# Import required modules
import cv2
import matplotlib.pyplot as plt

# Read the image
image = cv2.imread(r'/content/image.jpg')

# Convert color image into grayscale image
img1 = cv2.cvtColor(image, cv2.COLOR_RGB2GRAY)

# Plot grayscale image using Matplotlib
# cmap stands for colormap
plt.imshow(img1, cmap='gray')

# Display the image
plt.show()
```
