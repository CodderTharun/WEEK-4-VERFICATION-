## Operating Systems Overview

An Operating System (OS) is software that manages hardware resources and provides services to applications.

### Functions of OS

1. **Process Management**
   - Creating and terminating processes
   - Scheduling processes

2. **Memory Management**
   - Allocating memory to processes
   - Virtual memory management

3. **File Management**
   - File storage and retrieval
   - Directory management

4. **I/O Management**
   - Managing input/output devices
   - Device drivers

### Process vs Thread

- **Process** - Independent program execution with its own memory
- **Thread** - Lightweight unit of execution within a process
- Threads share memory within a process
- Multiple processes are isolated from each other

### Process States

1. **New** - Process created
2. **Ready** - Waiting for CPU
3. **Running** - Executing on CPU
4. **Waiting** - Waiting for I/O or event
5. **Terminated** - Process finished
