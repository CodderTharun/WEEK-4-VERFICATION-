# Database Management Systems - Week 1

## DBMS Introduction

A Database Management System (DBMS) is software for storing, retrieving, and managing data.

### Types of Databases

1. **Relational (SQL)**
   - Table-based structure
   - MySQL, PostgreSQL, Oracle, SQL Server

2. **NoSQL**
   - Document-based (MongoDB)
   - Key-value (Redis)
   - Column-family (Cassandra)

3. **Graph Databases**
   - Neo4j for relationship data

### Relational Database Concepts

- **Table** - Collection of rows and columns
- **Row/Record** - Single entry in a table
- **Column/Attribute** - Field with specific data type
- **Primary Key** - Unique identifier for each row
- **Foreign Key** - Reference to primary key in another table

### Database Operations

- **CRUD Operations:**
  - Create (INSERT)
  - Read (SELECT)
  - Update (UPDATE)
  - Delete (DELETE)

### SQL Basics

```sql
SELECT * FROM Students WHERE age > 20;
INSERT INTO Students VALUES (1, 'John', 22);
UPDATE Students SET age = 23 WHERE id = 1;
DELETE FROM Students WHERE id = 1;
```
