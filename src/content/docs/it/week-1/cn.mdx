# Computer Networks

## Introduction to Computer Networks

A computer network is a set of computers connected via communication channels.

### Network Types

1. **LAN (Local Area Network)**
   - Limited geographical area
   - High data transfer rate
   - Low latency

2. **WAN (Wide Area Network)**
   - Large geographical area
   - Lower data transfer rate
   - Higher latency

3. **MAN (Metropolitan Area Network)**
   - Between LAN and WAN
   - City-level coverage

### OSI Model

The OSI (Open Systems Interconnection) model has 7 layers:

1. **Physical Layer** - Cables, signals
2. **Data Link Layer** - MAC addresses, switching
3. **Network Layer** - IP addresses, routing
4. **Transport Layer** - TCP, UDP
5. **Session Layer** - Session management
6. **Presentation Layer** - Data encryption, compression
7. **Application Layer** - HTTP, FTP, SMTP
