# Data Structures & Algorithms - Week 5

## Dynamic Programming

Dynamic Programming (DP) solves problems by breaking into overlapping subproblems.

### Optimal Substructure

Solution to problem depends on solutions to subproblems.

### Overlapping Subproblems

Same subproblems solved repeatedly.

### Approaches

1. **Top-Down (Memoization)**
   - Recursion with caching
   - Solve subproblems on demand

2. **Bottom-Up (Tabulation)**
   - Iterative approach
   - Build solution bottom-up

### Classic DP Problems

1. **Fibonacci**
   - Recursive: O(2ⁿ)
   - DP: O(n)

2. **0/1 Knapsack**
   - Optimize item selection

3. **Longest Common Subsequence**
   - Compare two sequences

4. **Longest Increasing Subsequence**
   - Find longest ordered subsequence

5. **Coin Change**
   - Minimum coins for amount
