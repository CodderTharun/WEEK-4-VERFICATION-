# Operating Systems - Week 5

## Security

OS provides security mechanisms to protect resources.

### Access Control

- **Authentication** - Verify user identity
- **Authorization** - Determine permissions
- **Encryption** - Protect sensitive data
- **Audit Trails** - Track access and changes

### User and Group Management

- **Users** - Individual accounts
- **Groups** - Collection of users
- **Permissions** - Read, Write, Execute

### Process Security

- **Privilege Separation** - Minimum privilege principle
- **Sandboxing** - Isolated execution environment
- **Stack Overflow Protection** - Prevent buffer overflows
