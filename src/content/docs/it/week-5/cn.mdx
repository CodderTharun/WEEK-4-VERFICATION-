## Application Layer

Provides services directly to user applications.

### HTTP/HTTPS

Hypertext Transfer Protocol.

- **HTTP** - Insecure, port 80
- **HTTPS** - Encrypted with SSL/TLS, port 443
- **Methods**: GET, POST, PUT, DELETE, PATCH
- **Status Codes**: 2xx, 3xx, 4xx, 5xx

### Email Protocols

1. **SMTP** - Send emails (port 25)
2. **POP3** - Retrieve emails (port 110)
3. **IMAP** - Retrieve and manage emails (port 143)

### DNS (Domain Name System)

- Translates domain names to IP addresses
- Hierarchical naming system
- DNS queries and responses

### FTP (File Transfer Protocol)

- Transfers files between computers
- Active and passive modes
- Port 20/21

### SSH (Secure Shell)

- Secure remote access
- Encryption and authentication
- Port 22
