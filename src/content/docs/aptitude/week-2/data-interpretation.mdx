#  Multi-Data Analysis

## Introduction

This week advances data interpretation skills by analyzing multiple data sets simultaneously and drawing complex conclusions.

## Key Concepts

### 1. Multiple Data Representation

**Combined Data:**
- Multiple charts showing related information
- Tables with multiple columns and rows
- Time-series data across different categories

**Analysis Steps:**
1. Read each data set independently
2. Identify what each represents
3. Find relationships between data sets
4. Calculate aggregate values
5. Compare relative proportions

### 2. Complex Calculations

**From Multiple Sources:**
$$\text{Total} = \text{Data Set 1} + \text{Data Set 2} + ...$$

$$\text{Average of Multiple Sets} = \frac{\sum \text{All Values}}{\text{Total Count}}$$

**Percentage with Multiple Variables:**
$$\text{Percentage} = \frac{\text{Part}}{\text{New Total}} \times 100$$

### 3. Trend Analysis

**Comparing Over Time:**
- Identify growth or decline patterns
- Calculate rate of change
- Project future values
- Compare different categories' performance

### 4. Common Question Types

- "What is the total...?"
- "What percentage is X of Y?"
- "Which category increased the most?"
- "What was the average growth?"
- "Compare X with Y and Z"

## This Week's Topics

1. Interpreting combined bar and pie charts
2. Table and graph combinations
3. Multi-step calculations from data
4. Comparing data across multiple dimensions
5. Drawing valid conclusions

## Practice Problems

**Data Set:**
- Year 1: Product A = 100 units, B = 150 units
- Year 2: Product A = 120 units, B = 180 units

**Questions:**
1. Total units in both years?
2. Which increased more in%?
3. Average per product per year?
4. Projected Year 3 values (assuming same growth)?

## Key Calculations to Master

$$\text{Growth\%} = \frac{\text{New Value} - \text{Old Value}}{\text{Old Value}} \times 100$$

$$\text{Compound Growth} = \left(\frac{\text{Final}}{\text{Initial}}\right)^{1/n} - 1$$

## Tips for Success

- Write down all given information
- Organize data in tables if not provided
- Calculate step by step
- Double-check arithmetic
- Verify answers make logical sense
- Practice with diverse data combinations
- Build speed through repetition
