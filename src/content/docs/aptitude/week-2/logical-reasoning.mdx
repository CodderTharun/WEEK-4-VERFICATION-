# Analogies and Relationships

## Introduction

Analogies test your ability to recognize relationships between concepts. This week focuses on understanding different types of relationships and solving analogy problems.

## Key Concepts

### 1. Types of Relationships

**Synonym Relationship:**
- Smart : Intelligent
- Problem: If Smart, then ?

**Antonym Relationship:**
- Hot : Cold
- Problem: If Hot, then ?

**Part-to-Whole:**
- Finger : Hand
- Problem: If Finger, then ?

**Cause-and-Effect:**
- Rain : Wet
- Problem: If Rain, then ?

**Category Relationship:**
- Rose : Flower
- Problem: If Rose, then ?

**Function/Purpose:**
- Spoon : Eating
- Problem: If Spoon, then ?

### 2. Analogy Problem Format

**A : B :: C : ?**

Steps to solve:
1. Find the relationship between A and B
2. Apply the same relationship between C and ?
3. Check if your answer maintains the relationship

### 3. Common Relationship Patterns

- **Opposites**: Day - Night, Happy - Sad
- **Similarity**: Cat - Dog (both animals)
- **Degree**: Warm - Hot, Small - Tiny
- **Tool-Action**: Pen - Write, Thermometer - Measure
- **Quantity**: Dozen - 12, Score - 20

## Practice Problems

Solve the following analogies:

1. Lion : Roar :: Cow : ?
2. Teacher : School :: Doctor : ?
3. Morning : Evening :: Start : ?
4. Pen : Paper :: Brush : ?
5. Fish : Gills :: Human : ?

## Key Patterns to Remember

- A is to B as C is to D
- Find what connects A to B
- Match that connection with C and D
- Eliminate illogical options
- Consider multiple relationship angles

## Tips for Success

- Think of real-world connections
- Test each option against the relationship
- Avoid assumptions; use logic
- Practice varied relationship types
- Learn common analogy patterns
- Time yourself while solving
