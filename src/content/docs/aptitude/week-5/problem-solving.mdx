#  Integrated Scenarios & Mastery

## Introduction

Final week covers integrated problems combining multiple competencies - quantitative, logical, and strategic thinking.

## Key Concepts Review

### 1. Problem Type Integration

**Pure Quantitative:**
- Mathematical problems requiring only calculations
- Formulas and numerical reasoning

**Multi-Domain Problems:**
- Quantitative + Logical
- Quantitative + Strategic
- All three combined

**Real-World Scenarios:**
- Business cases
- Resource allocation
- Strategic decision-making
- Optimization under constraints

### 2. Advanced Problem Structures

**Cascading Problems:**
- Solution to part A enables part B
- Cumulative complexity
- Interdependencies

**Multiple Approaches:**
- Algebraic solution
- Logical/systematic approach
- Geometric/visual interpretation
- Numerical approximation

**Constraint Competition:**
- Multiple competing objectives
- Trade-off analysis
- Optimal balance finding

### 3. Strategic Problem Recognition

**Quick Classification:**
- Type identification: < 20 seconds
- Approach selection: < 30 seconds
- Execution: variable
- Verification: < 20 seconds

**Approach Selection Criteria:**
- Problem complexity
- Available techniques
- Time constraints
- Accuracy requirements

### 4. Advanced Verification

**Consistency Checks:**
- Answer magnitude reasonable?
- Units correct?
- Sign (positive/negative) correct?
- Boundary conditions satisfied?

**Alternative Verification:**
- Solve differently if possible
- Check with extreme values
- Verify with worked examples
