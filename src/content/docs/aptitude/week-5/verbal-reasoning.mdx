# Verbal Reasoning - Week 5: Passage Comprehension Mastery

## Introduction

Final week combines all reading skills with emphasis on accuracy and speed - critical for exam success.

## Key Concepts Review

### 1. Passage Mastery Strategies

**Pre-Reading (15 seconds):**
- Glance at title
- Note passage length
- Set expectations

**First Read (60-90 seconds):**
- Capture main idea
- Note structure
- Identify perspective

**Active Reading Techniques:**
- Margin notes on key points
- Mark author's main argument
- Circle supporting evidence
- Flag difficult concepts

### 2. Question Strategy

**Main Idea Questions:**
- Focus on first/last paragraphs
- Avoid too specific details
- Choose broad summary

**Supporting Detail Questions:**
- Locate information in passage
- Match exact wording carefully
- Avoid close-but-wrong options

**Inference Questions:**
- Based on passage only
- Logical connection required
- Not pure speculation

**Tone/Attitude Questions:**
- Word choice analysis
- Emotional tone indicators
- Author's stance

### 3. Passage Combination Analysis

**Compare passages when:**
- Multiple passages given
- Similarities requested
- Contradictions asked

**Contrast Analysis:**
- Different perspectives?
- Disagreements on facts?
- Opposing conclusions?

### 4. Speed Management

**Per Passage:**
- Reading: 1-2 minutes
- Questions: 1-1.5 minutes
- Total: 2-3.5 minutes

**Tracking:**
- Monitor time for each passage
- Adjust strategy if running behind
- Complete high-accuracy questions first

## This Week's Topics

1. Speed reading for comprehension
2. Accurate question analysis
3. Handling multiple passages
4. Critical perspective evaluation
5. Achieving consistency under pressure

## Exam Passage Types Covered

- Academic/Technical
- Narrative/Literary
- Argumentative/Opinion
- Social/Scientific

## Practice Metrics

- Accuracy: 85%+ target
- Speed: Complete in time
- Consistency: Same performance across types

## Multiple Passage Strategy

**With 2-3 Passages:**
1. Read all passage introductions
2. Identify easiest passage
3. Attempt questions systematically
4. Return to difficult passages
5. Review if time permits

## Tips for Success

- Focus on accuracy first
- Build reading speed gradually
- Practice diverse passage types
- Mock test under time pressure
- Review mistakes carefully
- Develop standardized approach
- Master consistency

Achieve excellence in verbal reasoning!
