# Game Theory and Strategic Decisions

## Introduction

This week introduces game theory and strategic decision-making - useful for understanding competitive scenarios and optimal strategies.

## Key Concepts

### 1. Basic Game Theory

**Players:** Decision makers involved
**Actions:** Choices available to each player
**Payoffs:** Outcomes/rewards for different action combinations
**Strategy:** Plan for which action to take given opponent's moves

### 2. Types of Games

**Cooperative vs. Non-Cooperative:**
- Cooperative: Players can make binding agreements
- Non-Cooperative: Players act independently

**Zero-Sum vs. Non-Zero-Sum:**
- Zero-Sum: One's gain = another's loss
- Non-Zero-Sum: More possibilities for mutual benefit

### 3. Nash Equilibrium

**Concept:**
- Each player's strategy is optimal given others' strategies
- No player can improve by unilateral change

**Finding:**
1. List all possible strategy combinations
2. Check if either player can improve by changing
3. Equilibrium: where neither can improve

### 4. Decision-Making Under Uncertainty

**Expected Value:**
$$E(X) = \sum (\text{Outcome} \times \text{Probability})$$

**Decision Rule:**
- Choose action with highest expected value
- Or minimize maximum loss (minimax)
- Or maximize minimum gain (maximin)

## Key Formulas

$$\text{Expected Value} = \sum (P_i \times V_i)$$

$$\text{Expected Loss} = \sum (P_i \times L_i)$$

## Strategic Principles

- Balance risk and reward
