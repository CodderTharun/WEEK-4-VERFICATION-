# Data Interpretation - Week 4: Correlation and Regression

## Introduction

This week introduces relationship analysis between variables - crucial for understanding cause-effect and predictive analysis.

## Key Concepts

### 1. Correlation

**Definition:** Measure of linear relationship between two variables

**Correlation Coefficient (r):**
$$r = \frac{\text{Covariance(X,Y)}}{\text{SD}_X \times \text{SD}_Y}$$

**Properties:**
- -1 ≤ r ≤ 1
- r = 1: Perfect positive correlation
- r = -1: Perfect negative correlation
- r = 0: No linear correlation

### 2. Types of Correlation

**Positive Correlation:**
- Both increase together
- Example: Study hours and grades

**Negative Correlation:**
- One increases, other decreases
- Example: Price and demand

**No Correlation:**
- No linear relationship
- Example: Shoe size and intelligence

### 3. Linear Regression

**Equation:**
$$y = a + bx$$

Where:
- y = dependent variable
- x = independent variable
- b = slope (rate of change)
- a = y-intercept

**Interpretation:**
- Slope shows how much y changes per unit of x
- Used for prediction and forecasting

### 4. Regression Analysis

**R-squared (R²):**
- Proportion of variance explained
- 0 ≤ R² ≤ 1
- Closer to 1: Better fit

## Key Formulas

$$b = r \times \frac{SD_Y}{SD_X}$$

$$a = \text{Mean}_Y - b \times \text{Mean}_X$$

$$\text{Predicted Y} = a + b \times X$$

## Critical Understanding

**Correlation ≠ Causation**
- Correlation indicates relationship
- Causation requires proof of mechanism
- Other factors may explain correlation
