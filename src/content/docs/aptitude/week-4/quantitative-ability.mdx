# Quantitative Ability - Week 4: Permutations, Combinations & Probability

## Introduction

These concepts form the mathematical foundation for counting and probability. Essential for competitive exams.

## Key Concepts

### 1. Permutations

**Definition:** Arrangements where order matters

**Formula:**
$$P(n,r) = \frac{n!}{(n-r)!}$$

**Special Cases:**
- All different: P(n) = n!
- Circular: (n-1)!
- With repetition: n^r

### 2. Combinations

**Definition:** Selections where order doesn't matter

**Formula:**
$$C(n,r) = \frac{n!}{r!(n-r)!}$$

**Properties:**
- C(n,0) = C(n,n) = 1
- C(n,r) = C(n,n-r)

### 3. Basic Probability

**Definition:**
$$P(\text{Event}) = \frac{\text{Favorable Outcomes}}{\text{Total Outcomes}}$$

**Properties:**
- 0 ≤ P(A) ≤ 1
- P(Certain) = 1
- P(Impossible) = 0

### 4. Probability Rules

**Addition Rule:**
$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

**Multiplication Rule:**
$$P(A \cap B) = P(A) \times P(B|A)$$
