# Logical Reasoning - Week 4: Logic Puzzles and Critical Thinking

## Introduction

This week focuses on solving complex logic puzzles and applying critical thinking to multi-layered reasoning problems.

## Key Concepts

### 1. Logic Puzzle Types

**Constraint Satisfaction:**
- Match attributes to subjects
- Use elimination
- Track constraints systematically

**Sequence Puzzles:**
- Arrange items based on clues
- Build valid ordering
- Check against all constraints

**Pattern Completion:**
- Complex multi-attribute patterns
- Identify hidden rules
- Apply to find solution

### 2. Problem-Solving Approach

**Step 1: Map Information**
- List all subjects/attributes
- Note which can be paired

**Step 2: Apply Constraints**
- If A then not B
- Either X or Y
- Eliminate impossibilities

**Step 3: Deduce Logically**
- Use "if-then" reasoning
- Find forced conclusions
- Update possibility matrix

**Step 4: Verify**
- Check solution against all clues
- Ensure no contradictions

### 3. Common Puzzle Formats

- **Days/Times:** Events on specific days
- **Colors/Attributes:** Matching properties
- **Rankings:** Ordering by criteria
- **Relationships:** Family/connections

### 4. Critical Thinking Questions

- Are all interpretations possible?
- What must be true given constraints?
- What can't be true?
- Are there logical chains?

## This Week's Topics

1. Solving constraint satisfaction puzzles

## Example Logic Puzzle

**Setup:**
- 3 people (A, B, C) buy different items (X, Y, Z)
- A didn't buy X
- B didn't buy Y
- C bought Z
- Who bought what?

**Solution:**
- C bought Z (given)
- B {X, Z} → B bought X (since C has Z)
- A {Y, Z} → A bought Y (since C has Z)

## Strategy Tips

- Create a matrix to track possibilities
- Mark what's definitely false
- Use logical chains
- One constraint can cascade to many conclusions
- Work systematically, not randomly

## Tips for Success

- Read puzzle carefully multiple times
- Extract all constraints precisely
- Use visual representations (matrices, diagrams)
- Apply constraints methodically
- Verify solution completely
- Practice diverse puzzle types
- Build pattern recognition skills

Master logic puzzles for advanced reasoning!
