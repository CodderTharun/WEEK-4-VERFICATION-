# Data Interpretation - Week 3: Statistical Analysis

## Introduction

This week introduces statistical concepts essential for analyzing and interpreting data scientifically.

## Key Concepts

### 1. Measures of Central Tendency

**Mean (Average):**
$$\text{Mean} = \frac{\sum \text{All Values}}{\text{Number of Values}}$$

**Median (Middle Value):**
- Arrange data in order
- If odd count: middle value
- If even count: average of two middle values

**Mode (Most Frequent):**
- Value that appears most often
- Can have multiple modes

### 2. Measures of Dispersion

**Range:**
$$\text{Range} = \text{Maximum} - \text{Minimum}$$

**Variance:**
$$\text{Variance} = \frac{\sum (x_i - \text{Mean})^2}{n}$$

**Standard Deviation:**
$$\text{SD} = \sqrt{\text{Variance}}$$

### 3. Data Distribution

**Normal Distribution:**
- Bell-shaped curve
- Mean = Median = Mode
- 68% data within 1 SD, 95% within 2 SD

**Skewed Distribution:**
- Positive skew: tail to right
- Negative skew: tail to left

### 4. Frequency Distribution

**Frequency Table:**
- Organizes data by intervals
- Shows frequency in each interval
- Cumulative frequency

## Key Statistical Formulas

$$z = \frac{x - \text{Mean}}{\text{SD}}$$ (Standardization)

$$\text{Coefficient of Variation} = \frac{\text{SD}}{\text{Mean}} \times 100$$

## Tips for Success

- Understand when to use which measure
- Mean for symmetric data, median for skewed
- SD shows spread of data
- Outliers affect mean and SD
