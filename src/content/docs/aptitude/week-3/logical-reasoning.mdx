# Logical Reasoning - Week 3: Syllogisms and Deduction

## Introduction

Syllogisms form the foundation of formal logic. This week teaches you how to recognize valid and invalid logical arguments.

## Key Concepts

### 1. Syllogism Structure

**Format:**
- Major Premise (statement 1)
- Minor Premise (statement 2)
- Conclusion

**Example:**
- All men are mortal (Major)
- Socrates is a man (Minor)
- Therefore, Socrates is mortal (Conclusion)

### 2. Types of Syllogisms

**Categorical Syllogisms:**
- Deal with categories and memberships
- Use All, Some, No relationships

**Conditional Syllogisms:**
- Use If-Then logic
- Based on conditions and outcomes

### 3. Valid vs. Invalid Arguments

**Valid Argument:**
- Conclusion logically follows from premises
- No logical fallacies
- Rules of logic are satisfied

**Invalid Argument:**
- Premises don't support the conclusion
- Contains logical fallacies
- Violates reasoning rules

### 4. Common Logical Fallacies

- **Undistributed Middle**: Middle term not distributed in both premises
- **Illicit Major/Minor**: Conclusion extends terms beyond premises
- **Exclusive Premises**: Both premises are negative
- **Negative-Affirmative Mix**: Negative premise with affirmative conclusion

## This Week's Topics

1. Understanding syllogistic structure
2. Identifying major and minor premises
3. Recognizing valid conclusions
4. Spotting logical fallacies
5. Complex multi-premise reasoning

## Practice Problems

Determine if conclusions are valid:

1. All dogs are animals / Max is a dog / Therefore, Max is an animal?
2. Some cats are white / Fluffy is white / Therefore, Fluffy is a cat?
3. No birds are insects / Eagles are birds / Therefore, Eagles are not insects?
4. All politicians are liars / Ram is not a politician / Therefore, Ram is not a liar?
5. If it rains, ground is wet / Ground is wet / Therefore, it rained?

## Reasoning Framework

**Valid Syllogism Checks:**
- Is the middle term distributed in at least one premise?
- Does conclusion extend terms appropriately?
- Are premises sufficient to support conclusion?
- Is there a logical fallacy?

## Tips for Success

- Visualize with Venn diagrams
- Identify all terms clearly (major, minor, middle)
- Check each validity rule
- Don't assume based on real-world knowledge
- Practice recognition of argument forms
- Build logical thinking skills systematically

Master syllogistic reasoning for logical excellence!
