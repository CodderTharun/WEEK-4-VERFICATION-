# Problem Solving - Week 3: Advanced Optimization

## Introduction

This week tackles complex optimization problems where you must maximize or minimize objectives while satisfying multiple constraints.

## Key Concepts

### 1. Linear Programming Basics

**Objective:**
- Maximize or minimize a linear function

**Subject to:**
- Linear constraints (inequalities or equations)

**Feasible Region:**
- Area satisfying all constraints
- Optimal solution at corner points

### 2. Problem Formulation

**Steps:**
1. Define decision variables
2. State objective function
3. List all constraints
4. Solve using appropriate method

**Example:**
- Maximize: Profit = 5x + 3y
- Subject to: x + y ≤ 10, x ≥ 0, y ≥ 0

### 3. Optimization Techniques

**Graphical Method:**
- Plot constraints
- Find feasible region
- Evaluate objective at corner points

**Algebraic Method:**
- Solve system of equations
- Check feasibility
- Compare objective values

### 4. Resource Allocation Problems

**Allocation Formula:**
$$\text{Total} = \text{Resource 1} \times \text{Allocation}_1 + \text{Resource 2} \times \text{Allocation}_2$$

## This Week's Topics

1. Formulating optimization problems
2. Graphical solution method
3. Algebraic solution approach
4. Resource allocation strategies
5. Sensitivity analysis

## Practice Problems

1. Maximize profit = 4x + 5y with constraints: x + y ≤ 8, x ≥ 0, y ≥ 0
2. A company producing A and B. A needs 2 hours, B needs 3 hours. 30 hours available. A gives profit 100, B gives 150. Maximize?
3. Production capacity: 100 units. Demand: A ≥ 20, B ≥ 15. Cost: reduce one increases other by 1.5. Minimize cost?
4. Budget 10,000. Investment A: return 20%, B: return 15%. Maximize return?
5. Three ingredients for recipe. Available: 50kg. Recipe uses: A 30%, B 50%, C 20%. Optimal quantities?

## Solution Template

**Step 1:** Define all variables and parameters
**Step 2:** Write objective function
**Step 3:** List all constraints with direction
**Step 4:** Identify feasible region
**Step 5:** Find optimal solution

## Tips for Success

- Be clear and organized in problem setup
- Double-check constraint formulation
- Consider all corner points (don't skip any)
- Verify solution satisfies all constraints
- Perform sensitivity analysis
- Practice diverse optimization types
- Build problem-solving confidence
