# Quantitative Ability - Week 3: Time, Work & Pipes

## Introduction

This week covers work-rate problems and pipes-cisterns - essential topics for time management and resource allocation problems.

## Key Concepts

### 1. Work Rate Basics

**Concept:**
- If A completes work in X days, A's work rate = 1/X per day
- Days = 1 / Rate

**Formulas:**
$$\text{Work Rate} = \frac{1}{\text{Days to Complete}}$$

$$\text{Combined Rate} = \text{Rate}_A + \text{Rate}_B$$

$$\text{Time Together} = \frac{1}{\text{Combined Rate}}$$

### 2. Multiple Workers Problems

**When A, B, C work together:**
$$\text{Combined Rate} = \frac{1}{a} + \frac{1}{b} + \frac{1}{c}$$

**Work Done = Rate × Time**

### 3. Pipes and Cisterns

**Inlet Pipe:** Fills the tank
**Outlet Pipe:** Empties the tank

**Net Rate = Inlet Rate - Outlet Rate**

**Time to Fill/Empty:**
$$\text{Time} = \frac{\text{Total Capacity}}{\text{Net Rate}}$$

### 4. Part-Time Problems

**When workers work for different periods:**
- Calculate individual contributions
- Sum to get total work done
- Set equation if needed

## Practice Problems

1. A completes a job in 8 days, B in 12 days. Together?
2. A can fill a tank in 3 hours, B empties it in 5 hours. Net time?
3. A, B, C working rates are 1/4, 1/6, 1/8 per day respectively. Together?
4. A works 2 days, B works 3 days, and complete the job. A works for X days to finish it alone?
5. Three pipes fill/empty with rates 1/2, 1/3, -1/4 per hour. Time to fill?

## Key Formulas

$$\text{Combined Rate} = \sum \text{Individual Rates}$$

$$\text{Work Done} = \text{Rate} \times \text{Time}$$

$$\text{Time to Complete} = \frac{\text{Total Work}}{\text{Rate}}$$

## Tips for Success

- Always express work rates as fractions
- Be careful with negative rates (outlet pipes)
- Set up equations clearly
- Verify by checking total work = 1
- Practice with various combinations
