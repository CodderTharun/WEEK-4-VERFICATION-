# Aptitude - Week 3

Week 3 moves into advanced territory with time-work problems, complex logical reasoning, and sophisticated data analysis.

## Week Overview

This week introduces more challenging concepts:

- **Quantitative Ability**: Time, work, and rate problems
- **Logical Reasoning**: Syllogisms and deductive reasoning
- **Verbal Reasoning**: Argument analysis and evaluation
- **Data Interpretation**: Statistical analysis basics
- **Problem Solving**: Complex multi-step optimization

## Learning Objectives

By the end of this week, you will:
- Solve time and work problems efficiently
- Understand logical syllogisms and deduction
- Evaluate arguments critically
- Perform statistical calculations
- Handle complex optimization scenarios

## This Week's Topics

- Quantitative Ability: Work Rate and Pipes/Cisterns
- Logical Reasoning: Syllogisms and Deduction
- Verbal Reasoning: Argument Evaluation and Bias
- Data Interpretation: Mean, Median, Mode, and Standard Deviation
- Problem Solving: Optimization with Multiple Constraints
