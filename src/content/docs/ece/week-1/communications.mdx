# Communications - Week 1

## Communication System

Sending information from transmitter to receiver.

### Components

1. **Source** - Information originator
2. **Transmitter** - Encodes message
3. **Channel** - Transmission medium
4. **Receiver** - Decodes message
5. **Destination** - Information user

### Signal Types

- **Analog** - Continuous amplitude
- **Digital** - Discrete values

### Modulation

Encoding information into carrier signal.

### This Week

- Communication basics
- Signal representation
- Noise and SNR
- Channel capacity
- Information theory basics

### Key Concepts

- Shannon's theorem
- Nyquist theorem
- Bandwidth requirements
- Signal-to-Noise Ratio (SNR)

- Design communication systems
- Calculate channel capacity
