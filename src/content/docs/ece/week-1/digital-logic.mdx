## Logic Gates

Basic building blocks of digital circuits.

### Basic Gates

1. **AND** - Both inputs high = high output
2. **OR** - Any input high = high output
3. **NOT** - Inverts input

### Derived Gates

1. **NAND** - AND followed by NOT
2. **NOR** - OR followed by NOT
3. **XOR** - Exclusive OR
4. **XNOR** - Exclusive NOR

## Boolean Algebra

Mathematics for logic circuits.

### Laws

- Commutative: A + B = B + A
- Associative: (A + B) + C = A + (B + C)
- Distributive: A(B + C) = AB + AC
- De Morgan's: (A + B)' = A'B'
