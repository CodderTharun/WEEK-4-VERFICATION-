### Explanation

- **Line 1:** `#include <stdio.h>` tells C to include a header file. This header lets us use input/output functions such as `printf()`.

- **Line 2:** A blank line. C ignores extra spaces and blank lines, but we use them to make the code easier to read.

- **Line 3:** `main()` is a special function. Your program starts running here. Any code inside the curly brackets `{}` will be executed.

- **Line 4:** `printf()` is a function used to output (print) text to the screen. In our example, it prints `"Hello World"`.

- **Line 5:** `return 0;` ends the `main()` function and sends a value back to the operating system. Returning `0` usually means the program executed successfully.

- **Line 6:** The closing curly bracket `}` marks the end of the `main()` function.

# Variables and Data Types

## Variables

Variables are containers for storing data values, like numbers and characters.

In C, variables must have a specific type, which tells the program what kind of data the variable can store.

- `int` — stores whole numbers (integers), such as `123` or `-123`
- `float` — stores numbers with decimals, such as `19.99` or `-19.99`
- `char` — stores a single character, such as `'a'` or `'B'`. Characters are surrounded by single quotes.

---

## C Variable Initialization

Once a variable is declared, we can store useful values in it. The first value we store is called the **initial value**, and the process is called **initialization**.

Initialization is done using the assignment operator `=`.

It is important to initialize a variable because a C variable contains a garbage value when it is declared without initialization.

We can also initialize a variable along with declaration.

---

## Rules for Naming Variables in C

- A variable name must only contain letters, digits, and underscores.
- It must start with an alphabet or an underscore only.
- It cannot start with a digit.
- No white space is allowed within the variable name.
- A variable name must not be a reserved word or keyword.
- The variable name must be unique in the program.

# Operators and Expressions

## Operators

Operators are used to perform operations on variables and values.

They are symbols that represent different kinds of operations such as mathematical, relational, logical, bitwise, and conditional computations.

The values or variables used with operators are called **operands**.

C divides operators into the following groups:

- Arithmetic operators
- Assignment operators
- Comparison operators
- Logical operators
- Bitwise operators

---

# Arithmetic Operators

Arithmetic operators are used to perform common mathematical operations.

| :---: | :---: | :--- | :---: |

| `*` | Multiplication | Multiplies two values | `x * y` |

# Assignment Operators

Assignment operators are used to assign values to variables.

The left-side operand of the assignment operator is a variable, and the right-side operand is a value.

The value on the right side must be of the same data type as the variable on the left side; otherwise, the compiler will generate an error.

Assignment operators can also be combined with other operators to perform multiple operations using a single operator. These are called **compound assignment operators**.

---

## Assignment Operators Table

| Operator | Name | Description | Syntax |
| :---: | :---: | :--- | :---: |
| `=` | Simple Assignment | Assigns the value of the right operand to the left operand | `a = b` |
| `+=` | Plus and Assign | Adds the right operand to the left operand and assigns the result | `a += b` |
| `-=` | Minus and Assign | Subtracts the right operand from the left operand and assigns the result | `a -= b` |
| `*=` | Multiply and Assign | Multiplies both operands and assigns the result | `a *= b` |
| `/=` | Divide and Assign | Divides the left operand by the right operand and assigns the result | `a /= b` |
| `%=` | Modulus and Assign | Assigns the remainder after division | `a %= b` |
| `&=` | AND and Assign | Performs bitwise AND and assigns the result | `a &= b` |
| `|=` | OR and Assign | Performs bitwise OR and assigns the result | `a |= b` |
| `^=` | XOR and Assign | Performs bitwise XOR and assigns the result | `a ^= b` |
| `>>=` | Right Shift and Assign | Performs right shift and assigns the result | `a >>= b` |
| `<<=` | Left Shift and Assign | Performs left shift and assigns the result | `a <<= b` |

# Logical Operators

Logical operators are used to combine two or more conditions or to reverse the result of a condition.

The result of a logical operation is always a Boolean value:

- `1` → True
- `0` → False

---

## Logical Operators Table

| `\|\|` | Logical OR | Returns `1` if at least one condition is true | `x < 5 \|\| x < 4` |
| `!` | Logical NOT | Reverses the result | `!(x < 5 && x < 10)` |

# Bitwise Operators

Bitwise operators are used to perform operations at the bit level.

The operands are converted into binary form, and operations are performed bit by bit.

Bitwise operations are useful for faster calculations and low-level programming.

---

## Bitwise Operators Table

| Operator | Name | Description | Syntax |
| :---: | :---: | :--- | :---: |
| `&` | Bitwise AND | Performs bit-by-bit AND operation | `a & b` |
| `\|` | Bitwise OR | Performs bit-by-bit OR operation | `a \| b` |
| `^` | Bitwise XOR | Performs bit-by-bit XOR operation | `a ^ b` |
| `~` | Bitwise Complement | Flips all bits | `~a` |
| `<<` | Left Shift | Shifts bits to the left | `a << b` |
| `>>` | Right Shift | Shifts bits to the right | `a >> b` |

---

# Operator Precedence

When an expression contains multiple operators, C follows precedence rules to determine which operation should be performed first.

Operators with higher precedence are evaluated before operators with lower precedence.

---

## Common Operator Precedence Order

| Priority | Operators | Description |
| :---: | :--- | :--- |
| 1 | `()` | Parentheses |
| 2 | `*`, `/`, `%` | Multiplication, Division, Modulus |
| 3 | `+`, `-` | Addition, Subtraction |
| 4 | `>`, `<`, `>=`, `<=` | Comparison |
| 5 | `==`, `!=` | Equality |
| 6 | `&&` | Logical AND |
| 7 | `\|\|` | Logical OR |
| 8 | `=` | Assignment |

---
