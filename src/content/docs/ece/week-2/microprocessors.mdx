## Instruction Types

### Data Transfer

- MOV - Move data
- LDA - Load accumulator
- STA - Store accumulator

### Arithmetic

- ADD, SUB - Add and subtract
- INR, DCR - Increment, decrement
- DAA - Decimal adjust

### Logical

- ANA, ORA, XRA - AND, OR, XOR
- CMA - Complement
- RLC, RRC - Rotate left/right
