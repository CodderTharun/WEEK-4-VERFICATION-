# Digital Logic - Week 2

## Combinational Circuits

Circuits with no memory, output depends only on current input.

### Design Process

1. Problem statement
2. Truth table
3. Boolean expression
4. Karnaugh Map simplification
5. Circuit implementation

## Karnaugh Map (K-map)

Graphical method for simplification.

### Properties

- Adjacent cells differ by one variable
- Group 1s to minimize expression
- Groups must be powers of 2

### Implementation

- 2x2, 3x3, 4x4 maps
- Prime implicants
- Essential prime implicants

### This Week

- Multiplexers and demultiplexers
- Encoders and decoders
- Half adder and full adder
- Comparators
- K-map minimization

### Practice

- Design combinational circuits
- Simplify using K-maps
- Verify designs
