# Communications - Week 2

## Analog Modulation

Encoding information into analog carrier.

### Amplitude Modulation (AM)

- Message modulates carrier amplitude
- Bandwidth: 2 × fm
- Susceptible to noise
- Simple and efficient

### Frequency Modulation (FM)

- Message modulates carrier frequency
- Bandwidth: 2(Δf + fm)
- Better noise immunity
- More bandwidth than AM

### Phase Modulation (PM)

- Message modulates carrier phase
- Similar to FM

### Modulation Index

- AM: μ = Am/Ac (depth of modulation)
- FM: β = Δf/fm (modulation index)
