# Communications - Week 4

## Noise

Unwanted signal degrading quality.

### Noise Types

- Thermal noise (Johnson noise)
- Shot noise
- Flicker noise
- Interference

### Signal-to-Noise Ratio

SNR = Ps/Pn (dB)

### Error Probability

Depends on:
- Modulation scheme
- SNR
- Channel characteristics

## Error Correction

### Error Detection

- Parity check
- Checksum
- CRC (Cyclic Redundancy Check)

### Error Correction

- Hamming code
- Reed-Solomon
- Turbo codes
- LDPC codes

- Noise analysis

- Shannon capacity
