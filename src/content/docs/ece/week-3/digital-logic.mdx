# Digital Logic - Week 3

## Sequential Circuits

Circuits with memory, output depends on current and previous inputs.

## Flip-Flops

Basic memory elements.

### SR Flip-Flop
- Set-Reset operation
- Race condition issue

### JK Flip-Flop
- Resolves SR limitations
- Toggle when J=K=1

### D Flip-Flop
- Data flip-flop
- Captures input at clock edge

### T Flip-Flop
- Toggle flip-flop
- Toggles on clock pulse

## Characteristics

- Setup time
- Hold time
- Propagation delay
- Clock-to-output delay
