# Microprocessors - Week 3

## Interrupts

Events that pause execution for immediate handling.

### Interrupt Types

- RESET - System reset
- INT 5.5, 6.5, 7.5 - Maskable interrupts
- TRAP - Non-maskable interrupt

### Interrupt Handling

1. Save program counter
2. Execute interrupt routine
3. Return to main program

### Interrupt Priorities

Different priority levels for handling multiple interrupts.

## I/O Interfacing

Connecting microprocessor to devices.

### I/O Modes

- Polling - Check device status
- Interrupt-driven - Device signals when ready
- DMA - Direct memory access

### Ports

- Parallel ports - Multiple bits simultaneously
- Serial ports - Bits sent one at a time
