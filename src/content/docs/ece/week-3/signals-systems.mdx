# Signals & Systems - Week 3

## Laplace Transform

Tool for analyzing linear systems.

### Properties

- Linearity
- Time shift
- Frequency shift
- Scaling
- Differentiation
- Integration

### Transfer Function

H(s) = Y(s)/X(s)

Characterizes system behavior.

## System Analysis

### Poles and Zeros

- Poles: Denominator zeros
- Zeros: Numerator zeros
- Pole-zero plots

### Stability

- All poles in left half-plane = Stable
- Poles on imaginary axis = Marginally stable
- Any pole in right half-plane = Unstable
