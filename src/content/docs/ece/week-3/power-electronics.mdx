# Power Electronics - Week 3

## PWM (Pulse Width Modulation)

Control technique using variable width pulses.

### PWM Parameters

- Frequency (switching frequency)
- Duty cycle - Ratio of ON to OFF time
- Resolution - Bits in PWM

### Motor Speed Control

Varying voltage/frequency controls speed.

## AC Motor Control

### Speed Control

- Pole changing
- V/f control
- VVVF (Variable Voltage Variable Frequency)

### Starting Methods

- DOL (Direct On Line)
- Star-Delta
- Soft starters
- VFD (Variable Frequency Drives)
