# Communications - Week 3

## Digital Modulation

Encoding digital information onto carrier.

### ASK (Amplitude Shift Keying)

- Binary 0: Low amplitude
- Binary 1: High amplitude
- Susceptible to noise

### FSK (Frequency Shift Keying)

- Binary 0: f₀
- Binary 1: f₁
- Better noise immunity

### PSK (Phase Shift Keying)

- Phase shift encodes bit value
- BPSK: 2 phase levels
- QPSK: 4 phase levels

### Performance

- Bit error rate
- Bandwidth efficiency
- Power efficiency
