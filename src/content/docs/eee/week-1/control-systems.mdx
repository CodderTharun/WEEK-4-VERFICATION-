# Control Systems - Week 1

## Open-Loop vs Closed-Loop

### Open-Loop
- No feedback
- Simple but inaccurate
- Susceptible to disturbances

### Closed-Loop
- Feedback control
- Accurate reference tracking
- Disturbance rejection

## Block Diagram

System representation using blocks and signals.

## Transfer Function

H(s) = Output/Input

Characterizes system behavior.
