# Mechanical Engineering - Week 3 Answer Keys

## Thermodynamics - Air Cycles

**Q1: Diesel Cycle**
```
Compression ratio higher than Otto
Heat addition at constant pressure
Efficiency: η = 1 - (1/r) × [rγ - 1]/[(γ)(rc - 1)]
rc: Cutoff ratio = V3/V2
Better compression → Higher efficiency
```

**Q2: Rankine Cycle (Steam)**
```
4 processes: Isentropic expansion, Condensation, Pumping, Boiler
Thermal efficiency depends on pressure ratio
Superheat improves cycle efficiency
Reheat in turbine improves performance
```

## Mechanics of Materials - Dynamics

**Q1: Material Testing**
```
Tensile test: Yield strength, Ultimate stress
Hardness test: Brinell, Vickers, Rockwell
Fatigue test: S-N curve
Impact test: Charpy, Izod
```

**Q2: Fatigue Analysis**
```
Endurance limit: σe (stress amplitude)
Goodman equation: 1/N = σm/σult + σa/σe
Miner's rule: Linear damage accumulation
Design for infinite life
```

## Machine Design - Optimization

**Q1: Optimization Framework**
```
Objective: Minimize cost or weight
Constraints: Strength, deflection, vibration
Method: Calculus, Lagrange multipliers
Trade-off analysis: Multiple objectives
```

**Q2: Vibration Analysis**
```
Natural frequency: f = (1/2π)√(k/m)
Damping ratio: ζ = c/(2√(km))
Response: Resonance at natural frequency
Isolation: Mount on springs or dampers
```

## Fluid Mechanics - Advanced

**Q1: Compressible Flow**
```
Mach number: M = V/c (speed of sound)
Stagnation properties: Constant along streamline
Choked flow: Maximum flow at sonic conditions
Isentropic relations for pressure/density
```

**Q2: Turbine Design**
```
Specific speed: Ns = N√Q/H^0.75
Classification: Pelton, Turgo, Crossflow
Efficiency: Varies with water head
Cavitation: Limit on suction head
```

## Manufacturing - Advanced

**Q1: CNC Programming**
```
G-codes: Motion commands
M-codes: Machine functions
Coordinate system: XYZ positioning
Tool compensation: Radius/length
```

**Q2: Surface Finish**
```
Ra: Average roughness
Rz: Maximum roughness depth
Achieved by: Tool geometry, Feed rate, Spindle speed
Measurement: Profilometer
```
