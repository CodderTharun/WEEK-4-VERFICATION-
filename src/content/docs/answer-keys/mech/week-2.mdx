# Mechanical Engineering - Week 2 Answer Keys

## Thermodynamics - Cycles

**Q1: Carnot Cycle**
```
4 reversible processes: 2 isothermal + 2 adiabatic
Max efficiency = 1 - Tc/Th
Theoretical maximum for any heat engine
No real engine achieves Carnot efficiency
```

**Q2: Otto Cycle (SI Engine)**
```
4 strokes: Intake → Compression → Power → Exhaust
Compression ratio: r = Vmax/Vmin
Efficiency: η = 1 - 1/r^(γ-1)
Higher ratio → Better efficiency
```

## Mechanics of Materials - Advanced

**Q1: Torsion of Shafts**
```
Maximum shear stress: τ = Tr/J
J: Polar moment of inertia
Angle of twist: φ = TL/GJ
Circular shaft: J = πd⁴/32
```

**Q2: Combined Loading**
```
Bending + Shear + Torsion together
Von Mises equivalent stress: σe = √(σx² + 3τxy²)
Failure by Von Mises criterion
Safe design: σe ≤ σallow
```

## Machine Design - Selection

**Q1: Bearing Selection**
```
Load rating: L10 life (10⁶ revolutions)
Bearing life: L = (C/P)³ × 10⁶
C: Bearing capacity
P: Applied load
Selection from manufacturer catalogues
```

**Q2: Spring Design**
```
Helical spring: k = Gd⁴/8D³N
Rate and natural frequency
Stress: τ = 8PD/πd³
Pre-load requirements
```

## Fluid Mechanics - Applications

**Q1: Pump Selection**
```
Head = P/(ρg) + V²/2g + z
Power = ρgQH/η
Affinity laws for operating conditions
Pump curves: Flow vs Head
```

**Q2: Pipe Flow**
```
Friction factor: f = 64/Re (laminar)
Major loss: hL = f(L/D)(V²/2g)
Minor loss: hL = KV²/2g
Total head requirement
```

## Manufacturing - Processes

**Q1: Forming Processes**
```
Rolling: Reduce thickness
Forging: Shape at high temperature
Drawing: Reduce diameter
Stamping: High-speed forming
```

**Q2: Casting Defects**
```
Porosity: Trapped air bubbles
Shrinkage: Volume reduction on cooling
Segregation: Uneven composition distribution
Prevention: Design, Temperature control
```
