# Mechanical Engineering - Week 1 Answer Keys

## Thermodynamics

**Q1: First Law of Thermodynamics**
```
ΔU = Q - W
Internal energy change = Heat added - Work done
For ideal gas: ΔU depends only on temperature
Sign convention: Q>0 (added), W>0 (done by system)
```

**Q2: Heat Engine Efficiency**
```
η = W / Qin = 1 - Qout/Qin
Carnot efficiency: ηCarnot = 1 - Tc/Th
Real engines < Carnot efficiency
```

## Mechanics of Materials

**Q1: Stress and Strain**
```
Stress σ = F/A
Strain ε = ΔL/L
Young's Modulus: E = σ/ε
Yield strength: Maximum before permanent change
```

**Q2: Bending Moment**
```
M = EI × d²y/dx²
Flexural stress: σ = My/I
Maximum at outer fibers
Design for safe stress levels
```

## Machine Design

**Q1: Design for Safety**
```
Factor of Safety = Ultimate Strength / Design Stress
Typical FOS: 2-5 depending on application
Fatigue considerations for cyclic loading
Material selection critical
```

**Q2: Shaft Design**
```
Torque transmission: τ = GIp/L
Combined stress: Von Mises stress
Diameter calculation: d = ∛(16T/πτs)
Critical speed: Whirling considerations
```

## Fluid Mechanics

**Q1: Continuity Equation**
```
A1V1 = A2V2
Q = A × V (volumetric flow rate)
For incompressible fluid
Q1 = Q2 → V2 = V1 × (A1/A2)
```

**Q2: Bernoulli's Equation**
```
P/ρg + V²/2g + z = constant
Energy conservation along streamline
Pressure + Dynamic + Potential = constant
Head losses via friction factor
```

## Manufacturing

**Q1: Machining Parameters**
```
Cutting speed: V = πDN/1000 (m/min)
Feed rate: f (mm/rev)
Depth of cut: d (mm)
Surface finish depends on tool geometry
```

**Q2: Tool Life**
```
Taylor's equation: VTⁿ = C
Higher speed → Shorter tool life
Optimal speed: Balance cost and efficiency
Regular tool change schedule
```
