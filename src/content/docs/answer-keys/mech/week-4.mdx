## Mechanics of Materials - FEA

**Q1: Finite Element Method**
```
Discretization: Divide domain into elements
Element equations: Stiffness matrix
Assembly: Global system matrix
Boundary conditions: Apply constraints
Solution: Displacement, Stress, Strain
```

**Q2: Von Mises Analysis**
```
3D stress state: σx, σy, σz, τxy, τyz, τzx
Principal stresses: σ1, σ2, σ3 from eigenvalues
Von Mises: σe = √(0.5×[(σ1-σ2)² + (σ2-σ3)² + (σ3-σ1)²])
Failure: σe > σyield
```
