# EEE Department - Week 2 Answer Keys

## Circuit Theory - Advanced

**Q1: Thevenin's Theorem**
```
1. Remove load
2. Find Vth = open circuit voltage at load terminals
3. Find Rth = resistance with sources deactivated
4. Thevenin equivalent = Vth in series with Rth
```

**Q2: Superposition Theorem**
```
Total response = Sum of responses for each source
- Turn off other sources (V-source→0, I-source→∞)
- Calculate response to one source
- Repeat and sum
```

## Control Systems - Analysis

**Q1: Bode Plot**
```
Magnitude plot: 20 log|H(jω)|
Phase plot: ∠H(jω)
Gain margin: 20 log(1/GM)
Phase margin: PM = 180° + ∠H(jωpc)
```

**Q2: Root Locus**
```
As K varies, poles move
Branches start at open-loop poles
End at zeros or infinity
Rules: Angle, magnitude, breakaway
```
