## Power Systems - HVDC

**Q1: HVDC Transmission**
```
Rectifier: AC → DC at sending end
Inverter: DC → AC at receiving end
Advantages: Long distance, Power control
Disadvantages: Cost, Harmonics
```

**Q2: FACTS Devices**
