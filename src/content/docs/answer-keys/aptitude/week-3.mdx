# Aptitude - Week 3 Answer Keys

## Quantitative Ability - Work & Time

**Q1: A completes in 8 days, B in 12 days, together?**
```
A's rate = 1/8 per day
B's rate = 1/12 per day
Combined = 1/8 + 1/12 = 3/24 + 2/24 = 5/24
Time = 24/5 = 4.8 days
```

**Q2: Tank fill times**
```
A fills in 3 hours, B empties in 5 hours
A's rate = 1/3, B's rate = -1/5
Combined = 1/3 - 1/5 = 5/15 - 3/15 = 2/15
Time = 15/2 = 7.5 hours
```

**Q3: Three workers A, B, C**
```
Rates: 1/4, 1/6, 1/8 per day
Combined = 1/4 + 1/6 + 1/8 = 6/24 + 4/24 + 3/24 = 13/24
Time = 24/13 ≈ 1.85 days
```

**Q4: Complex work scenario**
```
A works 2 days, B works 3 days
Setup equations: 2×(1/a) + 3×(1/b) = 1
Solve with constraint a = 6 for b
Result depends on specific values
```

**Q5: Three pipe rates**
```
1/2 + 1/3 - 1/4 = 6/12 + 4/12 - 3/12 = 7/12
Time = 12/7 ≈ 1.71 hours
```

## Logical Reasoning - Syllogisms

**Q1: "All men are mortal, Socrates is a man, therefore Socrates is mortal"**
```
Valid: Conclusion follows logically from premises
Middle term "man" distributed in major premise
```

**Q2: "Some cats are white, Fluffy is white, therefore Fluffy is a cat"**
```
Invalid: Undistributed middle term "white"
Possible that Fluffy is non-cat white thing
```

**Q3: "No birds are insects, Eagles are birds, therefore Eagles are not insects"**
```
Valid: Valid categorical syllogism
No → rejection relationship transfers
```

**Q4: "All politicians are liars, Ram is not a politician, therefore Ram is not a liar"**
```
Invalid: Cannot conclude from negative major premise
Affirming through negation fallacy
```

**Q5: "If it rains, ground is wet, Ground is wet, therefore it rained"**
```
Invalid: Converse fallacy
Ground could be wet for other reasons
```

## Verbal Reasoning - Argument Evaluation

**Argument: "Social media is harmful because teenagers spend too much time on it"**

**Q1: Main claim**
```
Answer: Social media is harmful
```

**Q2: Evidence provided**
```
Answer: Teenagers spend too much time on it
Critical evaluation: Insufficient evidence alone
```

**Q3: Logical fallacy**
```
Answer: Hasty generalization
Assumes all usage is harmful without evidence
```

**Q4: Strengthen argument**
```
Add: Mental health studies showing negative effects
Include: Long-term consequences data
```

## Data Interpretation - Statistics

**Dataset: 5, 8, 12, 15, 20, 22, 25**

**Q1: Mean**
```
Sum = 107, Count = 7
Mean = 107/7 ≈ 15.3
```

**Q2: Median**
```
Ordered: 5, 8, 12, 15, 20, 22, 25
Middle value (4th position) = 15
```

**Q3: Range**
```
Range = 25 - 5 = 20
```

**Q4: Standard Deviation (calculation)**
```
Deviations from mean squared and averaged
SD ≈ 7.2 (approximate)
```

## Problem Solving - Optimization

**Q: Maximize profit = 5x + 3y with constraints**
```
Constraints: x + y ≤ 10, x ≥ 0, y ≥ 0
Corner points: (0,0)→0, (10,0)→50, (0,10)→30
Maximum at (10, 0) = 50
```
