# Aptitude - Week 5 Answer Keys

## Quantitative Ability - Comprehensive

**Q1: LCS Problem**
```
String comparison using DP
Characters match → continue
Different → take max of two paths
Result: Longest subsequence length and sequence
```

**Q2: Maximum Subarray Sum**
```
Kadane's Algorithm:
max_current = max(element, max_current + element)
max_global = max(max_global, max_current)
Time: O(n), Space: O(1)
Answer: Largest sum contiguous subarray
```

**Q3: Complex integration**
```
Portfolio problem:
40% stocks @ 20% return
30% bonds @ 8% return
30% savings @ 3% return
Calculate total and after parameter change
```
