# Aptitude - Week 1 Answer Keys

## Quantitative Ability - Number Systems

**Q1: Prime Factors of 120**
```
120 = 2³ × 3 × 5
Prime factorization method:
120 ÷ 2 = 60
60 ÷ 2 = 30
30 ÷ 2 = 15
15 ÷ 3 = 5
5 is prime
```

**Q2: LCM of 4 and 6 (divisible by both)**
```
LCM = 12
Multiples of 4: 4, 8, 12, 16...
Multiples of 6: 6, 12, 18...
Smallest common = 12
```

**Q3: 25% of 480**
```
Answer: 120
25% = 1/4
480 × 0.25 = 120
```

**Q4: Ratio a:b = 3:4, b:c = 2:5, find a:c**
```
a:b = 3:4
b:c = 2:5
Make b same: a:b:c = 6:8:20
Therefore a:c = 6:20 = 3:10
```

**Q5: 3/8 as percentage**
```
Answer: 37.5%
(3/8) × 100 = 37.5%
```

## Logical Reasoning - Series

**Q1: Find missing number: 3, 7, 11, 15, ?, 23**
```
Answer: 19
Pattern: Each term increases by 4
Arithmetic progression with d = 4
15 + 4 = 19
```

**Q2: Complete series: 2, 4, 8, 16, ?**
```
Answer: 32
Pattern: Each term × 2
Geometric progression with r = 2
16 × 2 = 32
```

**Q3: Letter series: A, D, G, J, ?, P**
```
Answer: M
Pattern: Letters advance by 3 positions
A(1) → D(4) → G(7) → J(10) → M(13) → P(16)
```

**Q4: Mixed series: 1A, 2B, 3C, ?, 5E**
```
Answer: 4D
Numbers: 1, 2, 3, 4, 5 (sequential)
Letters: A, B, C, D, E (sequential)
```

**Q5: Fibonacci-like: 1, 1, 2, 3, 5, 8, ?**
```
Answer: 13
Pattern: Each term = previous two terms
5 + 8 = 13
```

## Problem Solving - Basic

**Q1: Work rate problem**
```
A completes in 4 days → Rate = 1/4
B completes in 6 days → Rate = 1/6
Combined: 1/4 + 1/6 = 5/12
Together = 12/5 = 2.4 days
```
