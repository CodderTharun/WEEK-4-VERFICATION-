# Aptitude - Week 4 Answer Keys

## Quantitative Ability - Permutations & Combinations

**Q1: Arrange 5 people in a line**
```
Answer: 5! = 120 ways
P(5,5) = 5!/(5-5)! = 120
```

**Q2: Select 3 from 10 items**
```
Answer: C(10,3) = 10!/(3!×7!) = 120 combinations
Order doesn't matter in selection
```

**Q3: Two heads in 3 coin tosses**
```
Total outcomes = 2³ = 8
Favorable (HHT, HTH, THH) = 3
Probability = 3/8 = 0.375
```

**Q4: Choose 2 aces from 52-card deck**
```
Total ways to choose 2 from 52 = C(52,2) = 1326
Ways to choose 2 aces from 4 = C(4,2) = 6
Probability = 6/1326 ≈ 0.0045
```

**Q5: Arrange 6 people around circular table**
```
Circular arrangement = (6-1)! = 5! = 120
Fix one position, arrange others
```

## Logical Reasoning - Logic Puzzles

**Puzzle Setup:**
```
3 people (A, B, C) buy items (X, Y, Z)
A ≠ X, B ≠ Y, C = Z
```

**Solution:**
```
C has Z (given)
B can have X or Z → B has X (C has Z)
A can have Y (A ≠ X, C has Z)
Answer: A-Y, B-X, C-Z
```

**Complex Puzzle - Days/Times:**
```
Use matrix elimination
Track constraints systematically
One constraint → cascade of deductions
Verify solution against all clues
```
