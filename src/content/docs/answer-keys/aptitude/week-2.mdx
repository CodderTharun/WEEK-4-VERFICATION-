# Aptitude - Week 2 Answer Keys

## Quantitative Ability - Profit & Loss

**Q1: Cost price 500, Selling price 625**
```
Profit = 625 - 500 = 125
Profit% = (125/500) × 100 = 25%
```

**Q2: Marked price 1000, Discount 20%, Find SP**
```
Discount = 20% of 1000 = 200
SP = 1000 - 200 = 800
```

**Q3: Successive discounts 10% and 20%, price 2000**
```
After 10%: 2000 × 0.9 = 1800
After 20%: 1800 × 0.8 = 1440
Final = 1440
Or: 2000 × 0.9 × 0.8 = 1440
```

**Q4: SI calculation**
```
P = 5000, R = 5% p.a., T = 4 years
SI = (5000 × 5 × 4) / 100 = 1000
Amount = 5000 + 1000 = 6000
```

**Q5: Shopkeeper - CP 60, Profit 25%**
```
SP = CP × (1 + Profit%)
SP = 60 × 1.25 = 75
```

## Verbal Reasoning - Inference

**Q1: "All plants need water. Rose is a plant."**
```
Inference: Rose needs water
Type: Logical deduction
Validity: Valid
```

**Q2: "It was 40°C yesterday."**
```
Possible inference: Today might be hot too
Type: Contextual inference
Validity: Likely but not certain
```

**Q3: "She completed the project ahead of schedule."**
```
Inference: She is efficient/organized
Type: Implicit inference
Validity: Reasonable based on evidence
```

- Year 1: A = 100, B = 150

```

(100+150+120+180) = 550
```

```
A: Growth = 20%
B: Growth = 20%
Same percentage but B has more absolute increase
```

```
Answer: 137.5
550 / 4 = 137.5
```

## Problem Solving - Multi-step

**Q1: Two numbers sum to 30, differ by 10**
```
x + y = 30
x - y = 10
Adding: 2x = 40 → x = 20
Subtracting: 2y = 20 → y = 10
Answer: 20 and 10
```

**Q2: Train speed distance time**
```
Distance = 240 km, Speed = 60 km/h
Time = D/S = 240/60 = 4 hours
```
