# IT Department - Week 5 Answer Keys

## Data Structures and Algorithms (DSA)

### Problem Set 1: Comprehensive Problems
**Q1: LCS (Longest Common Subsequence)**
```
DP[i][j] represents LCS length of first i and j characters
If s1[i] == s2[j]: DP[i][j] = DP[i-1][j-1] + 1
Else: DP[i][j] = max(DP[i-1][j], DP[i][j-1])
```

**Q2: Maximum Subarray Sum (Kadane's)**
```
max_current = max(num, max_current + num)
max_global = max(max_global, max_current)
Time: O(n), Space: O(1)
```

```
Answer: Cannot simultaneously guarantee:
```
