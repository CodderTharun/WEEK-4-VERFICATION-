# IT Department - Week 1 Answer Keys

## Data Structures and Algorithms (DSA)

### Problem Set 1: Arrays and Sorting
**Q1: Sort array using merge sort**
```
Answer: Merge sort implementation with O(n log n) complexity
- Divide array into halves recursively
- Merge sorted halves
```

**Q2: Find longest increasing subsequence**
```
Answer: Dynamic programming approach
- dp[i] = length of LIS ending at i
- Time: O(n²), Space: O(n)
```

### Problem Set 2: Linked Lists
**Q1: Reverse a linked list**
```
Answer: Iterative reversal
- Keep track of previous, current, next nodes
- Update pointers in reverse direction
- Time: O(n), Space: O(1)
```

**Q2: Detect cycle in linked list**
```
Answer: Floyd's cycle detection algorithm
- Use two pointers: slow and fast
- If they meet, cycle exists
```

## Database Management Systems (DBMS)

### Problem Set 1: SQL Queries
**Q1: Find employees earning above average**
```sql
SELECT name, salary FROM employees 
WHERE salary > (SELECT AVG(salary) FROM employees);
```

**Q2: Department-wise employee count**
```sql
SELECT dept_id, COUNT(*) FROM employees GROUP BY dept_id;
```
