## Question 1
1. The speed of a boat when travelling downstream is 32 km/h, whereas when 
travelling upstream it is 28 km/h. What is the speed of the boat in still water and 
the speed of the stream? 
- A) 30 km/h, 2 km/h 
- B) 28 km/h, 4 km/h 
- C) 32 km/h, 2 km/h 
- D) 30 km/h, 4 km/h

Correct Answer: A) 30 km/h, 2 km/h

Explanation: 
Speed of boat in still water 
= (1/2) × (32 + 28) 
= (1/2) × 60 
= 30 km/h 

Speed of stream 
= (1/2) × (32 − 28) 
= (1/2) × 4 
= 2 km/h

long will it take to cross a railway platform 165 m long? 

- A) 5 sec 
- B) 6 sec 
- C) 7.5 sec 
- D) 8 sec 

Correct Answer: C) 7.5 sec

Explanation: 
Speed of train 
= (132 × 5) / 18 m/s 
= 110 / 3 m/s 

Total distance covered 
= Length of train + Length of platform 
= 110 + 165 
= 275 m 

Time taken 
= Distance / Speed 
= 275 ÷ (110/3) 
= 275 × 3 / 110 
= 7.5 sec

## Question 9

9. The value of k for which the quadratic equation kx(x − 2) + 6 = 0 has equal roots is:

- A) 2
- B) 3
- C) 6
- D) −6

Correct Answer: C) 6

Explanation:

Given equation:

kx(x − 2) + 6 = 0

kx² − 2kx + 6 = 0

Here,

a = k

b = −2k

c = 6

For equal roots,

b² − 4ac = 0

(−2k)² − 4(k)(6) = 0

4k² − 24k = 0

4k(k − 6) = 0

k = 0 or 6

Since k ≠ 0 for a quadratic equation,

k = 6

## Question 10

10. Quadratic equation corresponding to the roots 2 + √5 and 2 − √5 is:

- A) x² − 4x − 1 = 0
- B) x² + 4x − 1 = 0
- C) x² − 2x − 1 = 0
- D) x² − 4x + 1 = 0

Correct Answer: A) x² − 4x − 1 = 0

Explanation:

Given roots:

α = 2 + √5

β = 2 − √5

Sum of roots

= α + β

= (2 + √5) + (2 − √5)

= 4

Product of roots

= αβ

= (2 + √5)(2 − √5)

= 4 − 5

= −1

Quadratic equation:

x² − (sum of roots)x + product of roots = 0

x² − 4x − 1 = 0

Hence, the required equation is

x² − 4x − 1 = 0

## Question 13

13. Which characteristic belongs to shallow copy?

- A) Creates separate memory allocation
- B) Creates completely independent objects
- C) Copies references of nested objects
- D) Avoids side effects completely

Correct Answer: C) Copies references of nested objects

Explanation:

A shallow copy copies the values of an object but does not create copies of nested objects.

As a result, changes made to shared nested objects may affect both the original and copied objects.

## Question 36

36. In ARP, when System A wants to find the MAC address of System B, how does it send the request and how does System B respond?

- A) Unicast request — Broadcast response
- B) Multicast request — Unicast response
- C) Broadcast request — Unicast response
- D) Broadcast request — Broadcast response

Correct Answer: C) Broadcast request — Unicast response

Explanation:

When System A knows the IP address of System B but not its MAC address, it sends an ARP Request to all devices on the local network using a broadcast message.

The device whose IP address matches the request identifies itself and sends an ARP Reply directly back to System A using unicast communication.

Thus, ARP uses a Broadcast Request followed by a Unicast Response.

- B) Continues transmitting and ignores the collision

## Question 39

39. Why does Simple Parity Check fail to detect even-numbered bit errors?

- A) Because it only checks the last bit of the data
- B) Because it uses multiplication instead of addition
- C) Because two flipped bits cancel each other out, keeping the parity count even
- D) Because it cannot handle data longer than 8 bits

Correct Answer: C) Because two flipped bits cancel each other out, keeping the parity count even

Explanation:

Parity checking works by verifying whether the total number of 1s remains even or odd.

If an even number of bits change during transmission, the parity remains unchanged.

This limitation makes parity checking unable to detect many even-numbered bit errors.

## Question 40

40. In CRC error detection, how does the receiver confirm that received data is error-free?

- A) By comparing the received data with a stored copy of the original
- B) By checking if the division of received data by the divisor gives zero remainder
- C) By recalculating the checksum and comparing it with the sent checksum
- D) By counting the number of 1s in the received data

Correct Answer: B) By checking if the division of received data by the divisor gives zero remainder

Explanation:

CRC uses binary polynomial division for error detection.

The sender appends CRC bits so that the transmitted data becomes exactly divisible by a predefined divisor.

At the receiver side, the same division is performed.

If the remainder is zero, the data is considered error-free; otherwise, an error is assumed to have occurred during transmission.

## Question 41

41. What is a deadlock in an operating system?

- A) A process finishes execution
- B) Multiple processes execute simultaneously
- C) Two or more processes wait forever for resources held by each other
- D) A process releases all resources

Correct Answer: C) Two or more processes wait forever for resources held by each other

Explanation:

A deadlock occurs when two or more processes are unable to proceed because each process is waiting for a resource currently held by another process.

Since none of the processes can obtain the required resource, all of them remain blocked indefinitely.

As a result, the system cannot make further progress until the deadlock is resolved.

## Question 45

45. In the Resource Request Algorithm of Banker’s Algorithm, after verifying Request ≤ Need and Request ≤ Available, what happens next?

- B) Resources are allocated temporarily and safety algorithm is checked
- C) Deadlock is declared immediately
- D) All resources are released
