# IT Department - Week 2 Answer Keys

## Data Structures and Algorithms (DSA)

### Problem Set 1: Trees and Graphs
**Q1: Binary tree traversals**
```
Pre-order (Root → Left → Right)
In-order (Left → Root → Right)
Post-order (Left → Right → Root)
Level-order (BFS)
```

**Q2: Graph DFS implementation**
```
Answer: Recursive/Stack-based traversal
- Mark visited nodes
- Explore all reachable nodes
- Time: O(V+E)
```

## Computer Networks (CN)

### Problem Set 1: Routing Protocols
**Q1: Dijkstra's algorithm routing**
```
Answer: Shortest path routing
- SPF (Shortest Path First)
- Used in OSPF
```

**Q2: IP Address Classes**
- Class A: 1.0.0.0 to 126.255.255.255
- Class B: 128.0.0.0 to 191.255.255.255
- Class C: 192.0.0.0 to 223.255.255.255

## Object-Oriented Programming (OOPS)

### Problem Set 1: Polymorphism
**Q1: Method overloading**
```
Answer: Same method name, different parameters
- Constructor overloading
- Function overloading
```

**Q2: Method overriding**
```
Answer: Redefine parent class method in child
- Achieves runtime polymorphism
- Uses virtual functions (C++) or @Override (Java)
```

## Operating Systems (OS)

### Problem Set 1: Memory Management
**Q1: Paging vs Segmentation**
- Paging: Fixed-size pages, No fragmentation
- Segmentation: Variable-size, External fragmentation

**Q2: Page Replacement Algorithms**
- FIFO: First In First Out
- LRU: Least Recently Used
- Optimal: Replace page not used for longest time

## Database Management Systems (DBMS)

### Problem Set 1: Normalization
**Q1: Types of Normal Forms**
- 1NF: No repeating groups
- 2NF: 1NF + No partial dependency
- 3NF: 2NF + No transitive dependency
- BCNF: Every determinant is a candidate key

**Q2: Functional Dependencies**
```
Answer: A → B means if two rows equal in A, 
they must be equal in B
```
